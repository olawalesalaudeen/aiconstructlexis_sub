# Plutus
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** Plutus-ben  

## State of the Field

The first Greek financial evaluation benchmark, assessing LLMs across six tasks: numeric NER, textual NER, question answering, extractive summarization, abstractive summarization, and topic classification using expert-annotated Greek financial datasets.
