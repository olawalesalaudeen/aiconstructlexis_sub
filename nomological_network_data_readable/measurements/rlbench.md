# RLBench
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, RLBench is characterized as a simulated benchmark for robot manipulation. It is specifically described as a "simulator of a Franka robot performing tabletop manipulation for a wide array of both prehensile and non-prehensile tasks" (HAMSTER: Hierarchical Action Models for Open-World Robot Manipulation). The single provided source uses this benchmark for two purposes: to generate trajectories and to evaluate manipulation-related path prediction. The paper details its use for generating a large dataset, specifically "1000 episodes for each of 81 robot manipulation tasks." Within this context, the instrument is referred to as a "popular robot manipulation benchmark."
