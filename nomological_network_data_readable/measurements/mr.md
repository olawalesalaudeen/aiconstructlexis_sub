## State of the Field

Across the provided literature, MR is consistently defined as a benchmark dataset for sentiment classification based on movie reviews. The prevailing usage is for evaluating binary sentiment, with definitions variously describing it as a task of "binary sentiment classification" or "polarity labeling." In practice, papers use MR to measure both the general behavior of `Text classification` and the more specific construct of `Sentiment analysis`. Its application is situated within a set of similar tasks, as one paper lists it alongside other sentiment classification benchmarks like SST-2 and CR. Additionally, some work employs MR as one of several "popular few-shot classification tasks." The characterization of MR as a movie review dataset for binary sentiment evaluation is uniform across all the provided sources.
