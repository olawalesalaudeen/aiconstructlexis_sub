# MFRC
**Type:** Measurement  
**Referenced in:** 5 papers  

## State of the Field

The Moral Foundations Reddit Corpus (MFRC) is characterized in the provided literature as a dataset of English-language Reddit posts. The corpus is consistently described as containing 16,123 posts sourced from 12 subreddits. One line of description focuses on the content domains, categorizing the posts into "US politics, French politics, and everyday moral life" ("InSerter: Speech Instruction Following with Unsupervised Interleaved Pre-training"). A complementary description frames MFRC as a "multi-label moral foundation corpus" annotated for six moral foundations, with each sample reviewed by three annotators ("REARANK: Reasoning Re-ranking Agent via Reinforcement Learning"). This latter work specifies that the dataset is used for "evaluating subjective multi-label classification and distribution alignment."
