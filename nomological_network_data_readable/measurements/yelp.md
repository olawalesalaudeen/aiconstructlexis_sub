# Yelp
**Type:** Measurement  
**Referenced in:** 9 papers  

## State of the Field

Across the surveyed literature, Yelp is consistently described as a dataset composed of user reviews from the Yelp platform, often characterized as containing opinions on restaurants, services, and businesses. The prevailing application of this dataset is for sentiment classification and analysis tasks, where it is used to evaluate downstream classifiers. For example, one study frames its use as a 5-way text classification task to assess the utility of synthetic data. While sentiment analysis is the most common focus, a smaller set of studies employs the dataset for different purposes. One paper uses Yelp for out-of-distribution evaluation of sentiment analysis models. Another study utilizes its collection of "casual reviews" to evaluate text origin detection methods on short-form natural language.
