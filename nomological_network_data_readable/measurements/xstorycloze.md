# XStoryCloze
**Type:** Measurement  
**Referenced in:** 10 papers  
**Also known as:** Xstory_cloze  

## State of the Field

Across the provided literature, XStoryCloze is predominantly characterized as a multilingual benchmark designed to evaluate story understanding and commonsense reasoning. The task is consistently operationalized by requiring a model to choose the correct ending for a four-sentence story. Its application for assessing multilingual and cross-lingual capabilities is a recurring theme, with one paper using it to "evaluate multilingual general and cross-lingual generation capabilities" ("LyapLock: Bounded Knowledge Preservation in Sequential Large Language Model Editing"). The benchmark is also frequently used to measure reasoning abilities; some papers explicitly state it measures "Commonsense understanding," while another describes it as a "widely used generic reasoning dataset" ("from Benign import Toxic: Jailbreaking the Language Model via Adversarial Metaphors"). A less common framing, from a single paper, also positions it as a tool for evaluating "multi-turn dialogues" ("CODI: Compressing Chain-of-Thought into Continuous Space via Self-Distillation").
