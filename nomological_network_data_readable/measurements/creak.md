# Creak
**Type:** Measurement  
**Referenced in:** 4 papers  
**Also known as:** CREAK  

## State of the Field

Creak is consistently described across the provided literature as a benchmark or dataset for fact-checking. There is some variation in how its purpose is framed: one definition specifies its use for "evaluating the verification of claims against a knowledge graph, using Wikidata," while another characterizes it as a benchmark that "assesses whether statements are supported by commonsense knowledge." This second framing is reinforced by sources that explicitly position Creak as a tool for evaluating "commonsense reasoning ability" in addition to fact-checking. In practice, papers use Creak to measure several constructs. It is most commonly used to assess `Commonsense understanding`, and is also reported as a measurement for `Faithfulness` and for `Evaluation` more broadly.
