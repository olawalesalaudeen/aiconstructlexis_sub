# OpenEQA
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the available information from a single paper, OpenEQA is characterized as a dataset designed for Embodied Question Answering. The dataset is defined by its structure, which provides instances composed of a question, a ground truth answer, and what is termed "associated episode memory from 3D environments." The source paper describes OpenEQA as offering "rich scenarios" for this task. This same work also notes that its own dataset construction process "builds upon" the OpenEQA dataset, positioning it as a foundational resource.
