# ToolACE
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, ToolACE is a dataset used for evaluation in the area of function-calling safety. The single source paper defines it as a collection of 11,300 rows intended to validate the generalization capability of models on these tasks. It is explicitly framed not as a standalone benchmark but as a supplementary or complementary evaluation set. For instance, one study utilizes it alongside MCIP-Bench to "further validate the generalization capability" ("TheLLMAlready Knows: EstimatingLLM-Perceived Question Difficulty via Hidden Representations"). Therefore, its documented use is as an additional resource for assessing model performance in safety-related function-calling scenarios.
