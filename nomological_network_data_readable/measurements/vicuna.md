# Vicuna
**Type:** Measurement  
**Referenced in:** 6 papers  

## State of the Field

Across the provided literature, Vicuna is predominantly described as a benchmark for evaluating the quality of open-ended text generation from language models. It is frequently used to assess model capabilities such as the helpfulness of responses, performance on instruction-following tasks, and the quality of conversational responses. One study also uses Vicuna as a measure of a model's commonsense knowledge. There is some variation in how the benchmark is operationalized: the most common description frames it as being based on "user-shared conversations" or "human-generated instructions from various tasks". A more specific characterization, found in one paper, defines it as a set of "80 challenging questions". Another source notes that the evaluation of conversational responses on this benchmark involves the use of GPT-4 ratings. In several papers, Vicuna is studied alongside other evaluation sets like WizardLM, Koala, and Self-instruct.
