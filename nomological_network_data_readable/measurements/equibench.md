# EquiBench
**Type:** Measurement  
**Referenced in:** 2 papers  

## State of the Field

A benchmark consisting of 2400 program pairs in Python, C, CUDA, and x86-64, designed to evaluate an LLM's ability to reason about program semantics via equivalence checking.
