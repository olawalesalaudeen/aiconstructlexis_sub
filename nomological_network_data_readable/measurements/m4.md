# M4
**Type:** Measurement  
**Referenced in:** 7 papers  

## State of the Field

The name M4 is used to refer to two distinct benchmarks in the provided literature, with one usage being more prevalent than the other. The most common framing across these papers describes M4 as a dataset for machine-generated text detection, specifically the "Machine-Made-Text-Multi-Domain-Multi-Generator" dataset containing documents from multiple LLMs across five domains. This version is used to evaluate detection approaches, with some work highlighting its multilingual nature for assessing "cross-lingual detection robustness." For instance, it is noted to include languages such as Chinese, Russian, and Urdu, and its Urdu subset has been used to augment training data. A separate and less frequent usage treats M4 as a benchmark for short-term time series forecasting. In this context, it is described as a testbed containing a "collection of marketing data in different sampling frequencies." This time-series version of the benchmark is used to measure model capabilities like In-context learning.
