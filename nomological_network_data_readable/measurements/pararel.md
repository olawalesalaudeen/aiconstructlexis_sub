# ParaRel
**Type:** Measurement  
**Referenced in:** 6 papers  

## State of the Field

Across the provided literature, ParaRel is consistently characterized as a dataset used to evaluate language models. It is specifically employed to measure the behavior of question answering and to study relational knowledge. The dataset is defined as containing factual knowledge, formatted as ⟨subject, relation, object⟩ triples. It is also described as a collection of 'paraphrased relational facts' with multiple query variations, intended for use in an 'independent multi-problem question answering setting'. One study notes that its use is 'consistent with other neuron-based methods'.
