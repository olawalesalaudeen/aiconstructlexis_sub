# XWinograd
**Type:** Measurement  
**Referenced in:** 4 papers  
**Also known as:** Xwinograd  

## State of the Field

Across the provided literature, XWinograd is consistently described as a multilingual or cross-lingual benchmark designed to test pronoun resolution. It is based on the Winograd Schema Challenge and is categorized as a "discriminative task". The provided papers frame what this task measures in slightly different ways. One definition presents it as a benchmark for "commonsense reasoning", and it is used in practice to measure the construct of `Commonsense understanding`. Another definition characterizes it as a task for testing "coreference and syntactic understanding", which aligns with its use in measuring `Coreference resolution`. The common element across these descriptions is the focus on pronoun resolution in diverse languages as the core evaluation method.
