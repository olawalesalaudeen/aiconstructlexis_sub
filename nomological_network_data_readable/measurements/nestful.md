# NESTFUL
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, NESTFUL is presented as a benchmark dataset designed to evaluate Large Language Models. Its specific purpose is to assess model performance on generating "nested (i.e. chained) sequences of API calls," as stated in "MaZO: Masked Zeroth-Order Optimization for Multi-Task Fine-Tuning of Large Language Models." The benchmark is designed for use in the domains of mathematical reasoning and generic coding. The evaluation is operationalized across five distinct performance dimensions. These metrics include F1 scores for both API functions (F1F) and their parameters (F1P), Partial Sequence Accuracy (PSA), Full Sequence Accuracy (FSA), and a Win Rate (WR).
