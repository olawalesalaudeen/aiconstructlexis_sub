# AGGREFACT
**Type:** Measurement  
**Referenced in:** 3 papers  
**Also known as:** AggreFact  

## State of the Field

Across the provided literature, AGGREFACT is consistently described as a benchmark dataset used to evaluate factual consistency within the task of summarization. The prevailing operationalization of the benchmark involves the CNN/DM and XSum datasets, a detail mentioned in multiple sources. One common definition further specifies that AGGREFACT provides "fine-grained annotations and summary-level labels across multiple domains" to facilitate its evaluation of factual consistency. In practice, it is used alongside other instruments like TRUE and SUMMAC, as noted in one paper's introduction: "tested on rich benchmark datasets, such as TRUE... SUMMAC... and AGGREFACT" (COVE:COntext andVEracity prediction for out-of-context images). Some work also refers to using specific data splits, such as "the AGGREFACT-FTSOTA split".
