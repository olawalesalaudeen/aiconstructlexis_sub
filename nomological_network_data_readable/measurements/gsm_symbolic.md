# GSM-symbolic
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

Based on the provided sources, GSM-symbolic is a benchmark dataset used to evaluate reasoning in large language models. It is defined as a collection of math word problems where names and numerical values are replaced with symbolic variables. The task for a model is to generate the correct symbolic expression as a solution. The available literature consistently presents this instrument as a tool designed to assess LLMs’ “mathematical reasoning skills.” One source also characterizes it as a “challenging symbolic reasoning benchmark.” Therefore, within this set of papers, the operationalization of mathematical reasoning is the successful generation of symbolic solutions to these modified word problems.
