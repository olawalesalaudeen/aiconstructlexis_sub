# ESConv
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided sources, ESConv is a benchmark dataset for emotional support conversations. It is defined as a collection of approximately 1,000 conversations between a 'help-seeker' and a 'supporter', with utterances annotated according to eight distinct support strategies. The dataset is used to measure text generation capabilities, specifically for evaluating performance in 'downstream response generation' within an emotional support context. One study reports using ESConv to demonstrate that its proposed framework 'excels at providing emotional support as well as diversifying support strategy selection'.
