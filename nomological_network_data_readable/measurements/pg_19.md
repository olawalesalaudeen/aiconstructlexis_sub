# PG-19
**Type:** Measurement  
**Referenced in:** 34 papers  
**Also known as:** PG19  

## State of the Field

Across the surveyed literature, PG-19 is consistently defined as a dataset or benchmark composed of long documents derived from Project Gutenberg books. The prevailing use of this instrument is to evaluate `language modeling` performance, with nearly all papers operationalizing this by measuring perplexity on these extended texts. This evaluation is frequently situated in the context of assessing `long-context processing` or `long-sequence language modeling`, as the dataset's book-length content provides what one paper calls a "comprehensive long-context benchmark." One source specifies the dataset consists of "over 28,000 freely-available Project Gutenberg books published prior to 1919."

While its primary function is measuring language modeling via perplexity, PG-19 is also used to assess a range of other capabilities. Several papers use it to evaluate `length generalization`, and a smaller set of studies employ it for more specific tasks, such as "benchmarking decoding speedup," assessing "model stability in streaming settings," or evaluating `continual learning`. The specific evaluation protocol varies, with some studies sampling a subset of documents, others using a validation set of 100 books, and at least one focusing on the first book of the test set. The constructs of `linguistic quality` and `long-form text generation` are also associated with PG-19, though less frequently detailed.
