# FB15k-237
**Type:** Measurement  
**Referenced in:** 7 papers  
**Also known as:** FB15K-237  

## State of the Field

Across the provided sources, FB15k-237 is consistently characterized as a benchmark dataset derived from Freebase. The prevailing usage of this instrument is to evaluate models on the task of knowledge graph completion, with papers also using it to measure link prediction and for general evaluation purposes. Multiple definitions agree on its construction, noting that it is a subset of the original FB15k dataset specifically created by removing inverse relations to prevent what is described as "test set leakage." A less common framing, found in one paper, describes it as "a large-scale knowledge graph with complex relational semantics" and uses it to assess graph partitioning. One source specifies that the dataset focuses on "common knowledge." In practice, it is often used in experiments alongside other benchmarks like WN18RR and YAGO3-10.
