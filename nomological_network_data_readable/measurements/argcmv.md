# ArgCMV
**Type:** Measurement  
**Referenced in:** 2 papers  

## State of the Field

A key point extraction benchmark dataset comprising ~12K arguments from real multi-turn debates on Reddit’s r/ChangeMyView, spanning ~3K topics, designed to evaluate long-context argument understanding and summarization.
