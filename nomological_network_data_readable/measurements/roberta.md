# RoBERTa
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, RoBERTa is characterized as a measurement instrument for safety evaluation, specifically for detecting harmful content. The single source paper defines it as a "harmfulness detection model" that functions as an "external safety evaluator." Operationally, it is used to assign a probability score to a given response, indicating the likelihood that the response is harmful. This work positions RoBERTa as a "state-of-the-art" tool for this purpose, employing it to assign "harmfulness probabilities to each response."
