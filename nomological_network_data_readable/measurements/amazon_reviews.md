# Amazon Reviews
**Type:** Measurement  
**Referenced in:** 5 papers  
**Also known as:** Amazon, Amazon Reviews dataset  

## State of the Field

Across the provided literature, the Amazon Reviews dataset is characterized as a collection of product reviews from various categories. Its prevailing use is to evaluate model performance, particularly for sentiment analysis on commercial user feedback. The dataset is also explicitly used as a measurement instrument for several classification behaviors, including text classification, node classification, and hierarchical text classification. One line of work describes adapting this dataset to include simulated user consultation texts, profiles, and interaction behaviors. This adaptation is intended to validate users’ search motivations, leveraging what one source calls the dataset's "real token text." The dataset is referred to as "widely used" in the provided materials.
