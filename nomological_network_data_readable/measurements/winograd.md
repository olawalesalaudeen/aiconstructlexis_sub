# Winograd
**Type:** Measurement  
**Referenced in:** 8 papers  

## State of the Field

Across the provided literature, Winograd is consistently described as a benchmark for coreference resolution that requires commonsense reasoning. The task is defined as involving pronoun disambiguation, where models must resolve ambiguous pronouns using both linguistic and commonsense knowledge. In evaluation settings, it is frequently grouped with other commonsense reasoning tasks, with papers listing it alongside benchmarks like PIQA, ARC, StoryCloze, and Winogrande. One paper provides a more specific formulation, describing it as a "language-modeling formulation of the WSC task from SuperGLUE" ("BERTs are Generative In-Context Learners"). The stated purpose of the instrument is to assess a model's capacity for pronoun resolution and commonsense inference.
