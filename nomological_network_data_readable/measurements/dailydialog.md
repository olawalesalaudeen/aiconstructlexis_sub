# DailyDialog
**Type:** Measurement  
**Referenced in:** 7 papers  

## State of the Field

Across the provided literature, DailyDialog is consistently presented as a dataset of human-written dialogues used for evaluating language models in conversational settings. The data is described as comprising "transcripts of conversations between humans" covering topics of daily life ("Confronting Reward Model Overoptimization with Constrained RLHF"). The prevailing use of the dataset is to assess model performance in dialogue generation, serving as a case study or general benchmark for "conversation scenarios" ("Compressed Context Memory for Online Language Model Interaction"). A more specific application is also noted in the provided data. One paper highlights that the dataset "includes 13,118 dialogues annotated with 7 emotions" and is used for tasks related to "emotion recognition and dialogue generation" ("Diagram-Driven Course Questions Generation"). This same work contrasts DailyDialog with other datasets that "mainly focus on emotion labels alone." Thus, while most papers in this set use DailyDialog to evaluate general dialogue generation capabilities, it is also operationalized as a resource for studying emotion in conversation.
