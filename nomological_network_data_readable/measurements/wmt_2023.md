# WMT 2023
**Type:** Measurement  
**Referenced in:** 3 papers  
**Also known as:** WMT’23  

## State of the Field

Across the provided literature, WMT 2023 is consistently defined as a benchmark or dataset from the 2023 Conference on Machine Translation, used to evaluate machine translation systems. The dataset is characterized by its inclusion of human-rated translations across multiple language pairs, with one paper specifically mentioning the use of "WMT’23 MQM ratings". Papers use this instrument to assess translation quality for language pairs such as English-German, Chinese-English, and pairs involving Russian, Czech, Ukrainian, and Hebrew. In practice, WMT 2023 serves as a test set for evaluating new methods, for example in the context of "fine-grained machine translation evaluation". Beyond direct system evaluation, the dataset is also used as a resource for studying methodological concerns, as one study sources "contaminated test examples" from WMT 2023 to analyze the impact of data contamination.
