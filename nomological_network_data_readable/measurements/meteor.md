# METEOR
**Type:** Measurement  
**Referenced in:** 8 papers  

## State of the Field

METEOR is consistently characterized across the provided sources as an automatic metric for evaluating text generation. It is frequently mentioned alongside ROUGE as a common evaluation tool. The papers describe its mechanism as word-based, incorporating features like synonymy, stemming, precision, and recall to assess quality. While its general purpose is generation evaluation, its specific application varies; one prevalent definition describes its use for assessing "generated legal responses against reference answers," while another source frames it as a metric for "translation quality."
