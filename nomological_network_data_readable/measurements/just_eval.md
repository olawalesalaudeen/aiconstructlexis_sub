# Just-Eval
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Across the provided literature, Just-Eval is described as an evaluation framework for assessing the outputs of large language models. It is consistently used to measure the construct of `Helpfulness`. The evaluation is multi-dimensional, with all sources noting that it assesses outputs along the aspects of helpfulness, clarity, factuality, depth, and engagement. As detailed in one paper, the framework is operationalized by using GPT-4 as a judge to score model outputs. This process involves scoring the output on a scale from 1 to 5 for each dimension, guided by what is referred to as the "official prompt" ("AfriMed-QA: A Pan-African, Multi-Specialty, Medical Question-Answering Benchmark Dataset").
