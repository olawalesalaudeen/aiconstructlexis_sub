# RAVEN
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** Raven  

## State of the Field

A non-verbal intelligence test, adapted as a visual reasoning benchmark (Zhang et al., 2019a), that requires identifying the missing element in a matrix of abstract patterns.
