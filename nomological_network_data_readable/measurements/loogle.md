# LooGLE
**Type:** Measurement  
**Referenced in:** 5 papers  

## State of the Field

LooGLE is consistently described across the provided literature as a benchmark or workload designed to evaluate large language models. The primary application of this instrument is to measure "Long-context processing," with definitions framing it as a tool for assessing both "long document question answering" and "long-context understanding." A defining feature highlighted in the sources is its use of prompts containing extensive text. As one paper notes, it "features extremely long documents... with over 24,000 tokens each" to assess both short- and long-dependency tasks ("Agri-CM3: AChinese Massive Multi-modal, Multi-level Benchmark for Agricultural Understanding and Reasoning"). In practice, it is cited as a workload alongside others for comparative system evaluations. The available data presents a unified view of LooGLE as an instrument for assessing an LLM's ability to derive answers from very long textual inputs.
