# AfriCOMET
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

Based on the provided data, AfriCOMET is a measurement instrument used to assess translation quality. It is defined as an enhanced version of the COMET metric specifically adapted for African languages. This adaptation is achieved by incorporating an "African-centric encoder, AfroXLM-R," and by training the model on African language data. The stated goal of this instrument is to improve translation evaluation performance for this specific language group. One paper also refers to a specific version of the instrument, "AfriCOMETv1.1-MTL".
