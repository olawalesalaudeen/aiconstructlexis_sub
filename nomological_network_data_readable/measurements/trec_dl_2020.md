# TREC-DL 2020
**Type:** Measurement  
**Referenced in:** 5 papers  
**Also known as:** TREC-DL20, TREC-DL'20, TREC-DL2020, TREC DL 2020  

## State of the Field

Across the provided literature, TREC-DL 2020 is consistently described as a benchmark dataset or test set from the TREC Deep Learning Track 2020. It is most commonly used to evaluate systems on information retrieval tasks, with several papers specifying its use for assessing performance in 'passage retrieval' and 'reranking' or 're-ranking'. One paper further specifies its application for evaluating 'LLM-based ranking systems on document relevance tasks'. The dataset is frequently mentioned alongside TREC-DL 2019, with one source noting it 'extend[s] TREC DL 2019 with additional queries and relevance annotations'. Two papers state its connection to MS MARCO, with one specifying it is 'derived from MS-MARCO-V1'. Papers characterize the benchmark in different ways; one describes it as an 'in-domain' evaluation dataset, while another notes that it is 'not reasoning intensive'.
