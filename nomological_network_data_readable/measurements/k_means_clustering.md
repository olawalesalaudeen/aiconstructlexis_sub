# K-means clustering
**Type:** Measurement  
**Referenced in:** 3 papers  
**Also known as:** K-Means clustering  

## State of the Field

Across the provided sources, K-means clustering is described as a statistical procedure used to group data points based on similarity. The prevailing application is to examine whether items form distinct groups in a high-dimensional space. For example, one paper uses K-means clustering on "LLM preference vectors" to group languages and identify "clusters of behavioral patterns." A different application involves using the method on "prompt embeddings" to see if they form natural groupings. This work mentions employing K-means clustering as one of two analytical methods, alongside "quantifying token importance." The procedure is implemented by running the algorithm with a pre-specified number of centers.
