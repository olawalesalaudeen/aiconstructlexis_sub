# Stanford Dogs
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

Stanford Dogs is consistently defined as a dataset for fine-grained visual classification, specifically for identifying 120 different breeds of dogs. It is explicitly used as a measurement instrument to assess the construct of "Fine-grained understanding". The dataset is situated as one of several within the larger FGVC benchmark, which focuses on Fine-Grained Visual Classification tasks. While its common application appears to be for classification, its use can be adapted. For instance, one paper notes that they "reformulate the task as a 4-way multiple-choice question," indicating some variability in how the benchmark is operationalized in practice.
