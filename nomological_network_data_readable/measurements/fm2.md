# FM2
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

Across the provided literature, FM2 is consistently characterized as a benchmark for the task of fact verification. The single available definition describes it as assessing "whether models can verify claims using retrieved evidence" (NOVA-63: Native Omni-lingual Versatile Assessments of 63 Disciplines). The benchmark is used to operationalize and measure multiple concepts within this context. For instance, one paper uses FM2 as a measure of `Adversarial robustness`. Another paper uses it more broadly for `Evaluation`, listing it as one of several English datasets for experimentation on the fact verification task.
