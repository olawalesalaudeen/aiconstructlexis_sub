# InfoSeek
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

InfoSeek is described in the provided literature as a visual question answering (VQA) dataset focused on evaluating knowledge-seeking abilities. The single available definition specifies that its questions require a two-step process: first identifying an entity in an image and then querying for a related fact about it. In practice, papers use InfoSeek to measure constructs such as `Generalization` and `Commonsense understanding`. One study situates it as a "popular KB-VR benchmark" (Knowledge-Based Visual Reasoning), employing it for evaluation alongside other instruments like A-OKVQA and OK-VQA ("FaithfulRAG: Fact-Level Conflict Modeling for Context-Faithful Retrieval-Augmented Generation"). This usage aligns with its core design of connecting visual content to external factual knowledge.
