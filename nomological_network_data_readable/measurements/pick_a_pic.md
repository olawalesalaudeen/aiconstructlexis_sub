# Pick-a-Pic
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Pick-a-Pic is characterized as a large-scale open dataset used to evaluate model outputs against human aesthetic and semantic judgments. The dataset is derived from a web platform that, as one paper notes, collects "real users’ creative intention" ("TIGeR: Unifying Text-to-Image Generation and Retrieval with Large Multimodal Models"). The primary application described for this instrument is to evaluate generation alignment with user preference. In practice, Pick-a-Pic is used to operationalize and measure the construct of Multimodal alignment.
