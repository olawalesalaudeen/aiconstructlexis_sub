# LongFact
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

Based on the available data, LongFact is characterized as a dataset designed to evaluate the factuality of language models, specifically within long-form generation tasks. It is defined as containing 250 prompts and is described as one of several "fact-seeking long-form generation datasets." The primary application shown in the provided sources is its use as a measurement instrument for Hallucination. In this capacity, it is explicitly identified as a "Factuality hallucination benchmark," operationalizing the assessment of factual accuracy in extended model outputs.
