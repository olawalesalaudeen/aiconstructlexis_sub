# LongBench-E
**Type:** Measurement  
**Referenced in:** 5 papers  

## State of the Field

LongBench-E is consistently used in the provided literature as a benchmark for evaluating model capabilities on long-context tasks. The prevailing description characterizes it as a collection of 13 real-world applications, with coding and few-shot learning cited as specific examples. A less frequent but more specific definition identifies it as the English-language subset of the broader LongBench benchmark. Across the surveyed work, its primary function is to measure `Long-context processing`, a use case mentioned in multiple papers. One paper also explicitly uses it to assess `Few-shot learning`. The benchmark is described as being "representative of many real-world long-context applications" ("LongMamba: Enhancing Mamba's Long-Context Capabilities via Training-Free Receptive Field Enlargement"), and researchers report conducting experiments on it to compare different models.
