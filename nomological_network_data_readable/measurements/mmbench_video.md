# MMBench-Video
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

MMBench-Video is described in the provided literature as a benchmark for evaluating multimodal understanding. The single available definition characterizes it as a "video-focused version of the MMBench benchmark." The primary and sole application mentioned across the sources is its use to measure "general video understanding." Papers report using MMBench-Video to assess model performance and establish new state-of-the-art results for this capability. The provided data does not contain information about other constructs or behaviors measured by this instrument.
