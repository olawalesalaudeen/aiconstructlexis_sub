# NTREX-128
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Across the provided sources, NTREX-128 is consistently identified as a dataset for evaluating document-level machine translation. One paper defines its specific application as measuring "English-to-Filipino translation" within the FILBENCH benchmark suite. The dataset is described as containing a "diverse set of test examples, ranging from documents." Another source broadens this scope, categorizing NTREX-128 as a multilingual benchmark alongside FLORES+, MAFAND-MT, and AfroLingu-MT. This paper notes that such benchmarks "have also incorporated African languages," suggesting a wider linguistic application than only Filipino.
