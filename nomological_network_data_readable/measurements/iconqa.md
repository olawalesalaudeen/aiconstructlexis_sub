# IconQA
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Across the provided literature, IconQA is consistently characterized as a dataset for visual question answering (VQA). The specific focus of this dataset is on 'diagram understanding', a task described as a form of VQA. In practice, IconQA is employed as a downstream dataset for fine-tuning Multimodal Large Language Models (MLLMs). It is used to evaluate visual reasoning capabilities and is sometimes studied alongside other VQA datasets such as ScienceQA. One source also identifies IconQA as a component task within the CMIT benchmark.
