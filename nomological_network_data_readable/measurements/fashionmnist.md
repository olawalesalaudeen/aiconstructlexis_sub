# FashionMNIST
**Type:** Measurement  
**Referenced in:** 3 papers  
**Also known as:** Fashion MNIST  

## State of the Field

Across the provided sources, FashionMNIST is consistently described as a dataset composed of Zalando's article images, with a training set of 60,000 examples and a test set of 10,000 examples. The dataset is used for the task of image classification. One paper specifies its role in a training pipeline, where it is used for fine-tuning, while the MNIST dataset is used for pretraining in the same context. Another source characterizes FashionMNIST as a 'drop-in replacement for MNIST'.
