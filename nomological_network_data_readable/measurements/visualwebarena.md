# VisualWebArena
**Type:** Measurement  
**Referenced in:** 6 papers  

## State of the Field

Across the provided literature, VisualWebArena is consistently defined as a benchmark environment for evaluating agents that operate on the web. Its primary stated purpose is to measure the performance of "multimodal web agents" or "vision-language agents" on the behavior of web navigation. The tasks within the benchmark are described as "realistic" and focus on "GUI interaction and task completion in browser environments" across platforms like classifieds, social media, and shopping sites. In addition to this central behavior, some work also uses VisualWebArena to measure the constructs of planning and commonsense knowledge. Researchers treat it as a "real environment" that can be adapted for specific experimental needs. For instance, studies report using it as a base for creating "targeted adversarial tasks" or employing specific subsets, such as a collection of "72 easy tasks" for testing.
