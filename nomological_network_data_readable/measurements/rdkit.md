# RDKit
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

Across the provided literature, RDKit is consistently described as a cheminformatics software toolkit or external tool. The data highlights two distinct applications for this instrument. One use is to calculate a variety of chemical properties of molecules; a specific paper mentions using it to compute drug-likeness (QED), synthetic accessibility (SA), and Lipinski's rule of five ("Fragment and Geometry Aware Tokenization of Molecules for Structure-Based Drug Design Using Language Models"). A different application involves using RDKit for verification, where it is employed to assess the validity of outputs from language models. In this role, it is used to "identify invalid SMILES generated by LLMs" by determining if they can be converted into valid molecules and by detecting specific chemical errors like valence or ring closure issues ("ConvSearch-R1: Enhancing Query Reformulation for Conversational Search with Reasoning via Reinforcement Learning").
