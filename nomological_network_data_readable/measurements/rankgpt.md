# RankGPT
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, RankGPT is characterized as a measurement method for ranking candidates within a Retrieval-Augmented Generation (RAG) pipeline. The single source defines it as a technique to order candidate papers based on specified criteria. Operationally, this is achieved by prompting the method with the desired ranking criteria, as one study notes when they "prompt it to rank papers according to the same three criteria" (COCO-Tree: Compositional Hierarchical Concept Trees for Enhanced Reasoning in Vision-Language Models). In the context of the available literature, RankGPT is presented as one of several tools, alongside methods like UPR, that are "designed for ranking candidates in the RAG pipeline."
