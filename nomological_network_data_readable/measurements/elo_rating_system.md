# Elo rating system
**Type:** Measurement  
**Referenced in:** 5 papers  

## State of the Field

The Elo rating system is consistently described across the provided literature as a method for calculating relative skill levels through pairwise comparisons, adapted from competitor-versus-competitor games. Its primary application in this context is to estimate model capability scores. The most common operationalization involves treating the evaluation process as a sequence of "model battles" to derive these scores. A more specific use case involves aggregating human judgments to create relative rankings, such as producing "fairness rankings of pruning methods." One paper notes that the system "dynamically adjusts ratings based on comparison outcomes and competitors’ relative strength" ("VisBias: Measuring Explicit and Implicit Social Biases in Vision Language Models"). As a measurement instrument, the Elo rating system is used to assess the construct of Faithfulness.
