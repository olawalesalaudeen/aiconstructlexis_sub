# RADAR
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Within the provided literature, RADAR is presented as an AI text detection system. Its defined purpose is to identify machine-generated text by leveraging distributional differences in language models. The system is also described as having robust cross-domain performance. In use, it is situated as one of several tools for this task, with one paper evaluating it as part of a "suite of prominent detectors" that also includes GPT-Zero, MAGE, and Binoculars.
