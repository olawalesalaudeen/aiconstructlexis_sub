# PLLuM-Align
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** PLLuM Preference Dataset  

## State of the Field

Polish preference dataset containing human-annotated response ratings, rankings, and multi-turn dialogues for evaluating alignment in Polish LLMs.
