# PromptEOL
**Type:** Measurement  
**Referenced in:** 2 papers  

## State of the Field

Prompt with Explicit One Word Limitation (PromptEOL) is a prompting technique used to generate compact embeddings by instructing the model with 'This sentence: “[text]” means in one word: '.
