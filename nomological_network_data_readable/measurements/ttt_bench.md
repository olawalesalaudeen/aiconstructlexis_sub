# TTT-Bench
**Type:** Measurement  
**Referenced in:** 2 papers  

## State of the Field

A benchmark consisting of four novel Tic-Tac-Toe-style games (oTTT, dTTT, cTTT, sTTT) designed to assess strategic, spatial, and logical reasoning through text-based board game questions asking for the best next move.
