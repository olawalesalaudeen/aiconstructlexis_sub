# COMETKIWI
**Type:** Measurement  
**Referenced in:** 6 papers  

## State of the Field

Based on the available literature, COMETKIWI is a learned metric used for evaluating machine translation. It is defined as an extension of the COMET evaluation metric, specifically trained on WMT Direct Assessment scores in a quality estimation style. The provided sources consistently use COMETKIWI to measure both "Machine translation" and "Translation quality". One paper positions it within a family of learned metrics, including COMET and xCOMET, that are reported to have improved MT evaluation. This improvement is attributed to the metric's ability to align "closely with human judgments" ("A Case Against Implicit Standards: Homophone Normalization in Machine Translation for Languages that use theGe’ez Script.").
