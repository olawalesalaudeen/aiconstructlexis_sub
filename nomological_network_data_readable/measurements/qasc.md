# QASC
**Type:** Measurement  
**Referenced in:** 11 papers  

## State of the Field

Across the provided literature, QASC is consistently described as a question-answering dataset used to evaluate model reasoning capabilities. The prevailing characterization is that it contains science-related questions requiring "multi-step" or "multi-hop" reasoning, which involves the integration of scientific facts from multiple sentences. Alongside this primary focus on scientific reasoning, a recurring theme is its use as a "commonsense reasoning dataset," and the provided data shows it is used to measure both "Question answering" and "Commonsense understanding." While most definitions focus on the reasoning task, one paper specifies its format as a "multiple-choice question answering benchmark." Researchers use QASC for performance evaluation, often alongside other datasets like StrategyQA and OpenBookQA, as well as for fine-tuning models. One study also adapts the dataset to "analyze the impact of knowledge retrieval on cross-lingual transfer."
