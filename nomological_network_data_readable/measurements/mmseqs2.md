# mmseqs2
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** MMSeqs2  

## State of the Field

A fast, sensitive protein search tool used as part of an analysis pipeline to efficiently identify homologous sequences in a database that are highly influential on a model's likelihood for a target protein.
