# MMHal-Bench
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, MMHal-Bench is a generative benchmark designed to evaluate hallucination in large multi-modal models. It is specifically used to assess the degree of hallucination across eight challenging question types. These types are enumerated as object attribute, adversarial object, comparison, counting, spatial relation, environment, holistic description, and a final category for others. The evaluation methodology described for this benchmark involves using GPT-4 to rate the severity of hallucination found in model responses.
