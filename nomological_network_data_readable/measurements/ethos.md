# ETHOS
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, ETHOS is a dataset used for the task of detecting hate speech. It is operationalized as a binary classification problem, with performance evaluated using the F1 score. The single source paper notes that this evaluation is conducted under "optimized prompts." The dataset is mentioned as being used alongside other benchmarks such as ArSarcasm and Liar, with one study reporting F1 scores across them (MultiSocial: Multilingual Benchmark of Machine-Generated Text Detection of Social-Media Texts).
