**Type:** Measurement

## State of the Field

In the provided data, Cosine similarity is framed as an objective evaluation procedure used to assess speaker similarity. This method operates by comparing speaker embeddings from synthesized speech against those from reference speech. The single source paper specifies that these embeddings are extracted using models such as ECAPA-TDNN or x-vectors. The sole application described is the quantitative measurement of how similar a synthesized speaker's voice is to a reference voice.
