# MoleculeNet
**Type:** Measurement  
**Referenced in:** 6 papers  

## State of the Field

Across the provided literature, MoleculeNet is consistently defined as a benchmark suite for molecular property prediction tasks. Its primary application is for evaluation, with one source specifying its use for classification evaluation and another describing it more broadly as a tool to assess model performance across diverse chemical properties. A common use of this benchmark, cited across multiple papers, is to operationalize and measure the construct of "Molecule understanding". The core task, as one paper describes, requires models "to answer about particular properties given the molecule." The suite is composed of various datasets, with papers citing examples such as BACE, BBBP, HIV, SIDER, ClinTox, MUV, Tox21, ESOL, and FreeSolv.
