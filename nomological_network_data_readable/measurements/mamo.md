# MAMO
**Type:** Measurement  
**Referenced in:** 4 papers  
**Also known as:** Mamo  

## State of the Field

Across the surveyed literature, MAMO is consistently described as a benchmark for evaluating mathematical modeling. The most common definition characterizes it as a tool for benchmarking modeling that requires the use of code solvers, with a specific focus on linear programming problems. A complementary framing, present in one paper, defines MAMO by its two optimization datasets, EasyLP and ComplexLP, which are used to evaluate LLM performance. In practice, MAMO is used to assess fine-tuned models, often alongside other benchmarks like NL4OPT, IndustryOR, and OptiBench. Models are reported to achieve "state-of-the-art performance" on MAMO, positioning it as an established instrument for measuring modeling capabilities.
