# VideoHallucer
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, VideoHallucer is characterized as a benchmark for evaluating video hallucination. Its stated purpose is to assess whether a model's generated responses are consistent with the content of a given video. The evaluation is structured across several categories, including "Basic, Hallucinated, and Overall." The single source paper in this set adopts VideoHallucer as one of two benchmarks for this purpose, using it alongside another instrument, EventHallusion, to measure video hallucination.
