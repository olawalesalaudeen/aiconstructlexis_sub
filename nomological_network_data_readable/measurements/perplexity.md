# Perplexity
**Type:** Measurement  
**Referenced in:** 17 papers  
**Also known as:** Perplexity Score, Sequence perplexity, Perplexity evaluation  

## State of the Field

Across the provided literature, the dominant framing of Perplexity is as a quantitative metric for evaluating language model outputs, though a distinct minority of papers use the name to refer to a commercial LLM search engine. In its most common usage as a measurement, it is operationalized to assess various aspects of text quality, such as fluency, coherence, and naturalness, with one paper describing it as a "lightweight proxy for naturalness" ("DICE: Structured Reasoning inLLMs throughSLM-Guided Chain-of-Thought Correction"). This general application is reflected in its use to measure constructs like Utility preservation, Faithfulness, and Imperceptibility. The evaluation procedure is sometimes specified as using a larger language model to calculate the perplexity of text generated by a smaller one. Beyond general quality assessment, Perplexity is also adapted into more specialized methods, including a "Perplexity Score" baseline for detecting pre-training data exposure and "sequence perplexity" as a value probing technique. In contrast, a smaller set of work treats "Perplexity" as a proper noun, identifying it as one of the "production LLM search engines" used as a target for evaluation.
