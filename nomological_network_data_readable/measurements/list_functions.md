# List functions
**Type:** Measurement  
**Referenced in:** 4 papers  
**Also known as:** List Functions, List Function  

## State of the Field

Across the surveyed literature, List Functions is consistently presented as a dataset or benchmark designed to evaluate reasoning capabilities in both humans and machines. The core task requires identifying or inducing a function that transforms an input list into a specified output list, with one paper noting the lists are of numbers. The most prevalent application of this instrument is to measure `Inductive reasoning`, a connection explicitly made across multiple papers. Several sources also describe its purpose as evaluating 'concept learning'. As one study notes, it "evaluates an LLM’s ability to induce rules that transform input lists into output lists" (LLMs Can Plan Only If We Tell Them). A less common framing, found in one paper, characterizes it as a 'Programming by Example (PBE) benchmark task'. The dataset is described as 'cognitive-science-inspired' and was originally designed to investigate how humans learn computable functions.
