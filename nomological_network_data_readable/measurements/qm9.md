# QM9
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

Across the provided literature, QM9 is consistently described as a benchmark for predicting molecular or quantum-chemistry properties. It is defined as a collection of over 130,000 small molecules with ground-state 3D structures. The primary task associated with this benchmark is computed property prediction, which is operationalized as a regression task. For example, one paper uses QM9 to evaluate performance on "several quantum chemistry measures such as HUMO, LUMO, and HUMO-LUMO gap" ('Hierarchical Graph Tokenization for Molecule-Language Alignment'). More broadly, QM9 is positioned as a measurement instrument used to assess a model's capacity for molecule understanding.
