# CommonsenseQA 2.0
**Type:** Measurement  
**Referenced in:** 3 papers  
**Also known as:** CSQA2  

## State of the Field

Across the provided literature, CommonsenseQA 2.0 is consistently described as a dataset used for evaluation. Its most prevalent application is to measure the construct of 'Commonsense understanding', a use case cited across multiple papers. A less common application, found in a single paper, is its use to measure 'Faithfulness'. The dataset is characterized as containing a 'large number of questions' in a multiple-choice format. One paper further specifies that these questions pertain to both 'entity knowledge and commonsense reasoning' (UniCoTT: A Unified Framework for Structural Chain-of-Thought Distillation). The definitions converge on its role as an instrument for assessing commonsense reasoning capabilities.
