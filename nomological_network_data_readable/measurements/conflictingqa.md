# ConflictingQA
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** CONFLICTINGQA  

## State of the Field

Benchmark dataset containing controversial yes/no web search queries and labeled support/refute web pages, used to evaluate model performance on debatable query summarization.
