# WSC
**Type:** Measurement  
**Referenced in:** 16 papers  

## State of the Field

Across the provided literature, WSC (Winograd Schema Challenge) is predominantly characterized as a benchmark for coreference resolution that requires pronoun disambiguation. The task is operationalized through multiple-choice questions, as noted in several sources. A recurring theme is that solving these questions necessitates commonsense reasoning, with some papers specifying "deep commonsense reasoning." While its primary application is measuring `Coreference resolution`, WSC is also used to evaluate `Commonsense understanding`. A smaller line of work also uses it to assess `In-context and few-shot learning`. One paper frames it more broadly as a general "reasoning benchmark" for evaluating downstream model performance.
