# True-False dataset
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, the True-False dataset is presented as a resource for training and evaluating factuality probes. It is defined as a labeled collection of factoid statements, each designated as either true or false. The dataset's content is structured around six disjoint topics: Cities, Inventions, Chemical Elements, Animals, Companies, and Scientific Facts, as described in "Beyond Position: the emergence of wavelet-like properties in Transformers". The sole purpose mentioned for this instrument is its use in the development and assessment of probes that target factuality.
