# OCR-VQA
**Type:** Measurement  
**Referenced in:** 5 papers  

## State of the Field

Across the provided literature, OCR-VQA is consistently presented as a benchmark for Visual Question Answering (VQA). The definitions from multiple papers agree that its primary function is to evaluate a model's ability to understand and reason about text present within images. This focus is further specified in source snippets, which describe the task as "Text Understanding VQA" and "text-in-image understanding." In practice, OCR-VQA is used to measure the behavior of `Visual question answering`, as it is listed among standard VQA datasets for experiments. Additionally, some work uses this benchmark to assess the construct of `Natural language understanding`.
