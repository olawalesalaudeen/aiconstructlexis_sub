# Food101
**Type:** Measurement  
**Referenced in:** 5 papers  

## State of the Field

Across the provided literature, Food101 is consistently characterized as a benchmark for evaluating image classification. The dataset is described as containing 101 categories of food dishes, with some sources specifying its use for "fine-grained classification." While its general purpose is to assess image classification, papers use it in several distinct evaluation settings. The most common definition states it is "used to test zero-shot class descriptions on common object categories." Other reported uses include evaluating "few-shot learning performance" and as a dataset for finetuning models. One paper also describes it as a "challenging image classification dataset."
