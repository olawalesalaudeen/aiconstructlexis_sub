# TheoremQA
**Type:** Measurement  
**Referenced in:** 9 papers  

## State of the Field

Across the provided literature, TheoremQA is consistently characterized as a benchmark used to assess mathematical reasoning in language models. The benchmark is defined as containing university-level STEM questions, with one source specifying it includes 800 problems from fields like math, physics, and engineering that require the application of mathematical theorems. Its primary function is to measure mathematical reasoning, a use case reinforced by multiple papers that group it with other mathematics-focused benchmarks such as MATH, GSM-Plus, SVAMP, and ASDiv. A less common application mentioned in one paper is its use as an out-of-domain test set.
