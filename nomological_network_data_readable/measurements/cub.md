## State of the Field

Across the provided sources, CUB is consistently identified as the Caltech-UCSD Birds dataset, often with the full name Caltech-UCSD Birds-200-2011. The prevailing usage of this instrument is as a benchmark for fine-grained classification tasks. It is most commonly defined as a dataset for "fine-grained image classification" that specifically features 200 species of birds. A similar framing describes it as a "fine-grained visual classification dataset." One paper also notes its application in "training and evaluating localization capabilities." The dataset is situated alongside other image classification benchmarks, as one source lists it with CIFAR-10, Food-101, and Flower.
