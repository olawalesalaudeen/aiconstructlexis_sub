# UNER
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, UNER is presented as a multilingual dataset for named entity recognition (NER). The single available definition specifies that it covers 13 languages and is used for evaluating cross-lingual transfer performance on NER tasks. In practice, it is used as a measurement instrument for the broader behavior of Information extraction. The source material mentions gathering UNER alongside other multilingual NER datasets, such as MasakhaNER2.0, for evaluation purposes.
