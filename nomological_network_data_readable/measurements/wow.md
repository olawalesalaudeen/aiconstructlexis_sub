# WoW
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, the Wizard of Wikipedia (WoW) dataset is characterized as a benchmark for evaluating knowledge-grounded dialogue. The single available definition specifies a conversational setting where one participant is given access to Wikipedia articles. The source paper, 'RAG-DDR: Optimizing Retrieval-Augmented Generation Using Differentiable Data Rewards', explicitly states that WoW is used for 'evaluation' in its work. This positions the dataset as an instrument for assessing model performance on dialogue tasks that require external knowledge.
