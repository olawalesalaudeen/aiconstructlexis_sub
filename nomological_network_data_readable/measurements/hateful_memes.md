# Hateful Memes
**Type:** Measurement  
**Referenced in:** 5 papers  
**Also known as:** HATEFUL MEMES  

## State of the Field

Across the provided literature, Hateful Memes is consistently framed as a measurement instrument for evaluating models on multimodal content. It is defined as both a "multimodal meme classification benchmark" and a "dataset for detecting hate speech," with the central task being to determine if an "image-text combination is hateful." The evaluation requires classifying whether a meme, including its image and text caption, "describes hateful intentions" or "promotes hateful sentiments." Papers in this set use Hateful Memes to operationalize and measure several constructs. It is most directly used to assess `Hate speech detection` and is also employed to evaluate the broader construct of `Harmlessness`. One definition specifies that the task requires "reasoning beyond simple unimodal signals," which aligns with its use in the literature to measure `Multimodal reasoning`.
