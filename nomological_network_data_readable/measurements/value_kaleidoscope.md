# Value Kaleidoscope
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Value Kaleidoscope is presented in the provided literature as a benchmark dataset for evaluating language models. It is defined as containing 31,000 human-authored moral dilemmas and hypothetical scenarios, each paired with specific ethical perspectives. The primary use of this benchmark is to evaluate whether a model's response to a scenario supports, opposes, or is neutral toward it from a particular viewpoint. In the context of its application, it is used to measure the construct of "Human preference alignment". One source also refers to it as a "trade-off steerability benchmark" intended for evaluating "steerable pluralistic models."
