# Minimal-pair ABX task
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** ABX discrimination task, Minimal-Pair ABX Tasks  

## State of the Field

A training-free, zero-shot evaluation framework where a model must determine if a test item X is representationally closer to item A or item B, used to measure the structure of embedding spaces.
