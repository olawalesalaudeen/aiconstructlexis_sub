# Progressive-hint prompting
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** Progressive-Hint Prompting  

## State of the Field

An evaluation method for scaling test-time compute that enables iterative interactions between a checker and an LLM, using previous answers as hints to guide the model.
