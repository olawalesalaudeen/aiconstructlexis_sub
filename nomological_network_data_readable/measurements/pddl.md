# PDDL
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, PDDL is a benchmark designed to evaluate LLM agents. Its stated purpose is to assess agent performance on planning tasks within PDDL-style environments. One paper specifies that the benchmark covers "diverse use cases in web, embodied, tool use and game scenarios" (AgentSquare: Automatic LLM Agent Search in Modular Design Space). In addition to its focus on planning, PDDL is also used as a measurement instrument for the construct of Decision-making.
