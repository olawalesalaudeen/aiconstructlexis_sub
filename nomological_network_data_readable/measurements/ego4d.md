# Ego4D
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided literature, Ego4D is characterized as a large-scale dataset composed of egocentric videos. The single paper in this set, "HourVideo: 1-Hour Video-Language Understanding," presents Ego4D as a source from which other benchmarks are curated. This is operationalized through the creation of the HourVideo benchmark, which is described as consisting of 500 videos selected from the Ego4D dataset. The source material from Ego4D used for this purpose is noted to cover 77 daily life scenarios.
