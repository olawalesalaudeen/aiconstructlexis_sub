# iCliniq
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** iCliniq dataset  

## State of the Field

Real-world medical consultation records from the iCliniq platform, used as an out-of-distribution test set to evaluate generalization in medical QA.
