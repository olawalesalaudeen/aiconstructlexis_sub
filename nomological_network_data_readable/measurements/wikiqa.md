# WikiQA
**Type:** Measurement  
**Referenced in:** 9 papers  

## State of the Field

Across the provided literature, WikiQA is consistently described as a benchmark for evaluating question answering (QA) capabilities. It is explicitly used to measure both general "Question answering" and more specific "Open-ended question answering". The application of the benchmark varies across studies, with several papers using it to assess answer selection performance from a set of candidate answers. A smaller line of work applies it to "closed-book QA," as noted in one paper ("Merge, Then Compress: Demystify Efficient SMoE with Hints from Its Routing Policy"). Another recurring use is for evaluating "spoken-query-based question answering (SQQA)," sometimes with synthesized speech inputs. A different framing focuses on the output format, where one study uses WikiQA for "sentence-level responses in calibration evaluation," observing that on this benchmark, "model responses are complete sentences" ("LitCab: Lightweight Language Model Calibration over Short- and Long-form Responses").
