# AMC
**Type:** Measurement  
**Referenced in:** 18 papers  
**Also known as:** AMC23, AMC2023  

## State of the Field

Across the surveyed literature, AMC is consistently used as a benchmark to evaluate the mathematical reasoning capabilities of models. It is defined as a dataset derived from the American Mathematics Competitions, with several papers specifically referencing the 2023 version. The benchmark is described as containing challenging, middle school-level math questions covering topics such as arithmetic, algebra, and geometry, and is used to test mathematical problem-solving skills. Performance on AMC is operationalized using metrics like the pass@1 score, as noted in one study. In evaluation settings, it is frequently used alongside other mathematical reasoning benchmarks, including AIME, MATH500, and MinervaMath. A single paper also lists it as a measure of commonsense knowledge, though this appears to be an isolated use within this dataset.
