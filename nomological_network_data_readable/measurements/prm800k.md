# PRM800K
**Type:** Measurement  
**Referenced in:** 5 papers  

## State of the Field

Across the provided literature, PRM800K is characterized as a dataset used to assess mathematical reasoning. The definitions describe its contents in complementary ways; some papers frame it broadly as containing "manually annotated step-by-step feedback on mathematical reasoning problems," while another provides more specific details, noting it "consists of 80,000 incorrect stepwise student solutions... each marked at the first error step." The problems themselves are sourced from the MATH dataset, which contains challenging competition-level math problems. The most common application described is for training models, specifically "process-supervised reward models" or a "step-level PRM." In line with its content and origin, the dataset is explicitly used to measure the construct of Mathematical reasoning.
