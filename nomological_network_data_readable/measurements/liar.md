# LIAR
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, LIAR is a dataset used for evaluating claim verification. The single source paper, "MultiSocial: Multilingual Benchmark of Machine-Generated Text Detection of Social-Media Texts", operationalizes this task as true/false classification. Performance on the dataset is assessed using the F1 score, with the paper noting an instance of "improving F1 score by 10.1 on LIAR dataset". This source presents LIAR as one of several "standard datasets" used for evaluation.
