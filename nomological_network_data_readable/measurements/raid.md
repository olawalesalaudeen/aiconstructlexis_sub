# RAID
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

RAID is consistently characterized across the provided sources as a dataset for evaluating machine-generated text detection. The prevailing description is that it is a multi-domain benchmark, with one paper specifically using its "Abstracts" and "Wiki" domains for evaluation. Its application for "cross-domain evaluation" is a stated purpose, and one study uses it as an "out-of-domain" dataset for this reason. A more detailed definition from one source specifies that RAID also includes adversarial examples and covers text from multiple models. In practice, researchers use RAID to evaluate their detection methods, often in conjunction with other datasets such as M4, MAGE, and RealDet.
