## State of the Field

Across the provided literature, IEMOCAP is consistently described as a benchmark database used to assess emotion-related tasks. The prevailing usage is for evaluating "emotion recognition from speech," a framing present in multiple sources. A related description characterizes it as a "multimodal dataset for emotion classification from spoken dialogue" and notes its application in evaluating "text-based emotion detection in long conversational sequences" (SUA: Stealthy Multimodal Large Language Model Unlearning Attack). In line with this, the dataset is used to measure the behavior of Classification. Researchers use IEMOCAP to report model performance, with one study noting accuracy improvements over other models. One paper specifies that the dataset contains 1,241 utterances and it is mentioned alongside other benchmarks like MELD and MultiWOZ 2.2.
