# Paired t-test
**Type:** Measurement  
**Referenced in:** 2 papers  
**Also known as:** Paired sample t-test  

## State of the Field

Statistical test comparing model perplexity on public versus private watermarked rephrasings of the same content to detect dataset membership, using a one-sided test for lower perplexity on public versions.
