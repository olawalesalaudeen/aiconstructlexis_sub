# WikiBio
**Type:** Measurement  
**Referenced in:** 4 papers  

## State of the Field

Based on the provided data, WikiBio is described as a benchmark dataset used for evaluating knowledge editing. The single available definition specifies its focus on "biographical information." The instrument is positioned as a way to operationalize and measure knowledge editing capabilities. For instance, one paper lists WikiBio among a set of datasets used to "edit different kinds of knowledge" ("Unlocking Efficient, Scalable, and Continual Knowledge Editing with Basis-Level Representation Fine-Tuning"). In this context, it is mentioned alongside other benchmarks like WikiDatarecent, WikiDatacounterfact, ConvSent, and ZsRE.
