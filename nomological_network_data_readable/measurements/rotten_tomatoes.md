## State of the Field

Across the provided literature, Rotten Tomatoes is consistently described as a dataset or benchmark for sentiment classification. One paper specifies that it is a dataset of movie reviews from the Rotten Tomatoes website used for binary sentiment classification ("Disentangling Biased Knowledge from Reasoning in Large Language Models via Machine Unlearning"). The primary behavior this instrument is used to measure is `Text classification`, as shown across multiple papers. Beyond this general task, it is also used to evaluate more specific methods, including "in-context learning and activation-vector methods" ("Iterative Vectors: In-Context Gradient Steering without Backpropagation") and "text classification with projected embeddings" ("Vector-ICL: In-context Learning with Continuous Vector Representations").
