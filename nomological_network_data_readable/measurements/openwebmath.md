# OpenWebMath
**Type:** Measurement  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, OpenWebMath is characterized as a large-scale mathematical web-text corpus. It is described as being filtered from CommonCrawl and containing 14.7B tokens of what one paper calls "high quality mathematical web text." The filtering process is specified to be based on "math strings, LATEXcontents and a math document classifier." The primary use documented for OpenWebMath, also referred to as OWM, is as a "seed dataset" or "seed corpus." For example, it is used as the foundation for generating a new synthetic dialogue corpus. The provided literature also positions it as a resource for "pretraining comparisons."
