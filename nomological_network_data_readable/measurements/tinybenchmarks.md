# TinyBenchmarks
**Type:** Measurement  
**Referenced in:** 4 papers  
**Also known as:** tinybenchmarks, tinyBenchmarks  

## State of the Field

Across the provided literature, TinyBenchmarks is consistently characterized as an evaluation suite designed for the efficient and rapid assessment of language models. It is defined as a collection of "small-scale datasets" or, more specifically, as "smaller, more efficient versions of popular benchmarks." The primary application described is measuring model performance on few-shot tasks, with one study citing its use for tasks such as "25-shot tinyARC, 5-shot GSM8K, 10-shot tinyHellaswag, 5-shot tinyWinoGrande." The recurring motivation for using this instrument is to achieve what one paper calls "efficient and low-cost evaluation." Another paper explicitly states it uses TinyBenchmarks "to further increase the efficiency of experiments," reinforcing its role in streamlined model assessment.
