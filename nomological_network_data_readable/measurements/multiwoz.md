# MultiWOZ
**Type:** Measurement  
**Referenced in:** 7 papers  

## State of the Field

MultiWOZ is characterized as a task-oriented conversational corpus, with one source describing it as a "Wizard-of-Oz corpus" for a booking system involving domains like hotels and restaurants. The provided sources show it is used for different evaluation purposes. The most prevalent framing, described as "widely used" in one source, is as a benchmark dataset for measuring dialogue state tracking performance across multiple domains and slot-value combinations. A different line of work uses MultiWOZ to evaluate first-order uncertainty quantification. In this application, the instrument is used to assess a model's ability to predict user satisfaction, which is operationalized by using the dataset's "average crowd-worker annotation as ground-truth." The sources also note the existence of different versions, such as MultiWOZ 2.4, which contains corrections and enhancements.
