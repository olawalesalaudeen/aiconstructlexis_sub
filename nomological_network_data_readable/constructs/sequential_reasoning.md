# Sequential reasoning
**Type:** Construct  
**Referenced in:** 4 papers  

## State of the Field

Based on the single source provided, sequential reasoning is defined as a latent ability to understand and reason over sequences of actions and states while maintaining a coherent interpretation across multiple steps. This capability is framed as a persistent challenge for large language models, which are reported to "struggle with multi-step sequential reasoning." The concept is discussed in the context of "long-horizon sequential tasks." The provided material associates limitations in sequential reasoning with other model weaknesses, such as those in compositional reasoning, planning, and length generalization. The data does not specify any particular instruments or tasks used to measure this construct.
