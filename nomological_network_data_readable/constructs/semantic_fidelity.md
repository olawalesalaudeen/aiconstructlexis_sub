# Semantic fidelity
**Type:** Construct  
**Referenced in:** 5 papers  

## State of the Field

Based on the provided data, semantic fidelity is a construct concerning the preservation of meaning in text that has been paraphrased or rewritten. The single source defines it as the extent to which a modified text "preserves the original meaning and core information of the source text" ("MUZO: Leveraging Multiple Queries and Momentum for Zeroth-Order Fine-Tuning of Large Language Models"). This concept is framed as a desirable quality for text generation, with one paper noting that "revised sentences should preserve the original semantics and exhibit a high degree of coherence to ensure text quality." The term appears to be used alongside "semantic similarity," as the source mentions a method "maintaining semantic similarity" in the same context as preserving fidelity. The provided materials describe this construct but do not offer information on how it is operationalized or measured.
