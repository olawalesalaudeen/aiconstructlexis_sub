# Cross-modal grounding
**Type:** Construct  
**Referenced in:** 3 papers  
**Also known as:** Audio-visual grounding, GUI action grounding  

## State of the Field

The ability to correctly align and reference information between visual and textual modalities, ensuring that answers are grounded in the actual content of the image.
