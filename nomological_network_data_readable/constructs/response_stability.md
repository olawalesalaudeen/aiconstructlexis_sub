# Response stability
**Type:** Construct  
**Referenced in:** 2 papers  
**Also known as:** Output stability  

## State of the Field

The degree to which a model's predicted bias remains consistent across different surface realizations of the same question, such as paraphrases, indicating robustness to superficial prompt variations.
