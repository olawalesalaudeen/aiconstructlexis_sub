# Information integration
**Type:** Construct  
**Referenced in:** 5 papers  

## State of the Field

Across the provided literature, 'Information integration' is consistently framed as a latent ability of models to combine information from multiple sources. However, the scope of this integration is described in two distinct ways. One perspective treats it as an inter-document capability, defining it as the ability to "combine evidence across multiple passages to determine relevance in multi-hop retrieval settings" ('Attention in Large Language Models...'). This framing is associated with tasks that require a model to understand relationships between separate documents. A different framing presents it as a more general, intra-context ability to "combine and utilize information from different parts of a linguistic context... particularly across sentence and clause boundaries" ('Middo...'). In this view, information is seen as hierarchically distributed across scales like words and sentences, and a central consideration is a model's "information integration window." Thus, the construct is used to describe model capabilities ranging from multi-hop reasoning across documents to processing dependencies within a single text.
