# Multi-image reasoning
**Type:** Construct  
**Referenced in:** 7 papers  
**Also known as:** Multi-image comprehension  

## State of the Field

Across the surveyed literature, multi-image reasoning is most commonly defined as the latent ability to integrate information from several images to answer questions or infer relations, requiring cross-image comparison and synthesis. This capability is frequently contrasted with the more common focus on single-image visual understanding. A related, less frequent framing is "multi-image comprehension," which one paper defines as the ability to identify shared or consistent elements across visual inputs. To operationalize and measure this construct, papers use several benchmarks, including Mantis-Eval, MMMU, and BLINK. The construct is studied alongside knowledge-based question answering and in-context and few-shot learning. One paper notes that performing this type of analysis necessitates "extended context capacity" from the model.
