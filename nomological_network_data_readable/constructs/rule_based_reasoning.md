# Rule-based reasoning
**Type:** Construct  
**Referenced in:** 4 papers  

## State of the Field

The provided sources consistently define rule-based reasoning as a latent ability to understand, integrate, and apply multiple abstract rules simultaneously within a complex task. These rules are specified to include counting, color logic, and shape constraints, as well as arithmetic and geometric rules requiring step-by-step planning. This construct is described as "deep, abstract, rule-based reasoning" and is sometimes situated within constrained spatial pathfinding environments. It is operationalized through tasks like SPaRC, which is reported to require "path-finding, counting, segregation, and logic involving colors and shapes in a single task". The capacity for precise rule following, a component of this construct, is noted as a challenge for large language models. Rule-based reasoning is also studied alongside mathematical reasoning, with which it shares this emphasis on precise rule following, and is related to inductive reasoning.
