# Authenticity
**Type:** Construct  
**Referenced in:** 6 papers  

## State of the Field

The most prevalent definition of Authenticity across the provided literature is the extent to which generated text reflects the "meaning, content, linguistic patterns, and contextual appropriateness of the target community's language." This framing emphasizes that an aligned LLM should accurately mirror a specific population's communication. However, several other distinct uses appear in the data. A smaller line of work treats Authenticity as faithfulness or credibility relative to a source document, such as scientific content. Other papers apply the construct to agent behavior, defining it as an agent's alignment with the "nuanced characteristics of a real individual" or as a user simulator's semantic and stylistic similarity to real human utterances. A more distinct usage frames it as website credibility, focusing on the absence of deceptive practices like "keyword overuse or content duplication." The construct is operationalized in multiple ways depending on its definition. For instance, it is measured through Human evaluation alongside other qualities like readability. It is also measured computationally by comparing semantic and stylistic similarity using tools like SimCSE and style embeddings, or evaluated on a "binary scale" for the presence of deceptive tactics.
