# Referring expression comprehension
**Type:** Construct  
**Referenced in:** 3 papers  
**Also known as:** Referring, Referring ability  

## State of the Field

Across the provided literature, 'Referring expression comprehension' is a construct used to describe a model's ability to ground a reference in visual or temporal media, though its specific definition varies. The most prevalent framing defines it as the latent ability to identify a specific visual instance from a natural-language description, or as one paper puts it, "the process of localizing the specific instances described by natural language inputs" ("A Large-Scale Human-Centric Benchmark..."). A different line of work describes a similar "referring ability" as the capability to ground responses in spatial regions of an image indicated by *visual prompts* like boxes or points, rather than text. A third, distinct usage applies the concept to the temporal domain, defining it as the ability to "comprehend time information from user inputs and identify which part of a video the user is referring to" ("E.T. Bench..."). Thus, the construct is used to characterize different grounding abilities that vary in both the modality of the reference (text vs. visual prompt) and the domain of the target media (spatial images vs. temporal videos).
