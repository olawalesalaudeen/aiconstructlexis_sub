# Token efficiency
**Type:** Construct  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, token efficiency is defined as the degree to which a model produces accurate answers without unnecessarily long or verbose responses. The concept is framed as a balance between correctness and conciseness. It is operationalized with a specific formula, defined as "accuracy divided by response length." This measurement is used to track whether models "make efficient use of tokens rather than producing unnecessarily verbose traces." The source material suggests that token efficiency can be improved through training; for example, one finding indicates that increasing the number of long Chain-of-Thought training examples can lead to "a general gain in token efficiency" by simultaneously improving accuracy and reducing response length.
