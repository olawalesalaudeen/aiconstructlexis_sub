# Multi-task learning
**Type:** Construct  
**Referenced in:** 3 papers  

## State of the Field

Within the provided literature, multi-task learning is defined as the ability of a model to perform well on multiple, distinct tasks at the same time, often by sharing knowledge between them. This capability is framed in the context of performance on previously 'seen' tasks, and is explicitly contrasted with zero-shot generalization, which applies to 'unseen' tasks. The concept is discussed in relation to Large Vision-Language Models (LVLMs), with one study presenting a method that reportedly demonstrates 'superior multi-task learning capabilities'. The available data does not specify the particular measurement instruments or behaviors used to operationalize this construct.
