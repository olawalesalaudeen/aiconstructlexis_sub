# Coarse perception
**Type:** Construct  
**Referenced in:** 3 papers  

## State of the Field

Coarse perception is defined in the provided literature as a model's latent ability to perceive and describe high-level, general elements within an image. This construct is operationalized and measured using specific evaluation instruments. The papers in this set use the MMBench and MMStar benchmarks to assess this capability. The concept is used to characterize the performance of vision-language models, with one study noting that certain models demonstrate "strong coarse... perceptual abilities." This ability is also framed in contrast to fine-grained perception, distinguishing between general and detailed visual understanding.
