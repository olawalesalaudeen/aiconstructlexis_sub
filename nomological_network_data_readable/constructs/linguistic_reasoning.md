# Linguistic reasoning
**Type:** Construct  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, linguistic reasoning is defined as a latent ability to logically process and manipulate the structure and meaning of language. This construct is specified to include skills such as disentangling syntax from semantics and inferring entailments. The concept is discussed in the context of Vision-Language Models (VLMs), with one paper noting "a lack of robust linguistic reasoning processes in VLMs". This source further suggests that VLM pre-training, which relies on image-caption pairs, "causes catastrophic forgetting of linguistic reasoning". This deficit is highlighted by a comparison to Large Language Models (LLMs), which are reported to "often significantly outperform VLMs in language understanding".
