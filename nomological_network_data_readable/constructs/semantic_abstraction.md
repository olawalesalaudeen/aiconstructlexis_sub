# Semantic abstraction
**Type:** Construct  
**Referenced in:** 2 papers  
**Also known as:** Logical abstraction, Compression, Semantic generalization  

## State of the Field

The process by which a model develops more efficient, shared neural representations under capacity constraints, abstracting away language-specific features to encode generalized semantic content.

## Sources

**[PQR: Improving Dense Retrieval via Potential Query Modeling](https://aclanthology.org/2025.acl-long.661.pdf) (as "Compression")**
> The process by which a model develops more efficient, shared neural representations under capacity constraints, abstracting away language-specific features to encode generalized semantic content.

**[PQR: Improving Dense Retrieval via Potential Query Modeling](https://aclanthology.org/2025.acl-long.661.pdf) (as "Semantic generalization")**
> The model's ability to form generalized concept representations that are shared across languages, allowing semantic information to be concentrated and abstracted away from surface linguistic forms.

**[ReflecTool: Towards Reflection-Aware Tool-Augmented Clinical Agents](https://aclanthology.org/2025.acl-long.664.pdf) (as "Logical abstraction")**
> The ability to identify and operate on the underlying symbolic structure of a problem, separating it from its surface-level lexical representation.
