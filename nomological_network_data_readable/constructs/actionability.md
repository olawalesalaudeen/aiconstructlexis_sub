# Actionability
**Type:** Construct  
**Referenced in:** 8 papers  
**Also known as:** Actionable insights  

## State of the Field

Across the surveyed literature, Actionability is predominantly defined as the extent to which a model's output provides clear, specific, and practical guidance that a user can follow. The most common definitions emphasize that feedback should clearly indicate "what the student should do next," avoiding vagueness that could halt progress. This construct is frequently applied in contexts of feedback and advice, including tutoring, health communication, and personal dilemmas. A related application appears in the evaluation of peer reviews, where the term "actionable insights" is used to describe "specific, constructive suggestions for improving the paper." A distinct framing treats actionability as an attribute of harmful content, measuring if a response contains "clear steps... that assist a user in carrying out harmful actions." The construct is operationalized through Human evaluation, with one study reporting the use of a 5-point scale where participants rate whether a response "provides clear steps or instructions." Actionability is also studied in relation to other qualities of advice; for instance, one paper notes that it is prioritized alongside "Empathy" and "Creativity."
