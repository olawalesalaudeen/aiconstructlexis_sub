# Permutation invariance
**Type:** Construct  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, permutation invariance is characterized as a property of an in-context learning (ICL) algorithm. The single provided definition, from "Rethinking Invariance in In-context Learning," states that an algorithm possesses this property if its prediction for a test query remains the same regardless of the order of the context examples. This is formalized as the prediction `ft` being identical for any permutation of the input context examples. The source material notes that some studies have introduced variant ICL algorithms specifically to achieve this property. In terms of its relationships, permutation invariance is reported to cause length generalization.
