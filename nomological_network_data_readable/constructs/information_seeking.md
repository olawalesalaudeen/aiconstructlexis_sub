# Information seeking
**Type:** Construct  
**Referenced in:** 4 papers  
**Also known as:** Information-seeking ability  

## State of the Field

Across the provided literature, 'Information seeking' is framed in two distinct ways. The prevailing usage treats it as a proactive, latent capability of a model to identify its own knowledge gaps and gather necessary information by asking questions, especially in settings with uncertainty. This view defines it as the ability to "elicit missing details through questioning in order to make more informed decisions" ('MediQ...'). In this context, a successful model must decide when it has sufficient information to act. A contrasting and less common framing, from a single paper, defines 'Information seeking' not as a model capability but as a category of user query, specifically the task of responding to questions that "explicitly ask for factual information or explanations" ('Instruct-SkillMix...'). Operationalization of the construct reflects this division. The active capability is evaluated using the MEDIQ benchmark, which is designed to assess this "question-asking ability" ('MediQ...'), while the passive task framing is used to categorize instructions in datasets.
