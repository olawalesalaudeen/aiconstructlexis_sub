# Semantic coverage
**Type:** Construct  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, semantic coverage is defined as the degree to which a model-generated text captures the key topics, concepts, and meaning of a reference text. This construct is used to evaluate LLM-generated content, such as literature summaries and compositions, by comparing them against human-written counterparts. The work operationalizes this concept by using "commonly employed semantic similarity metrics and Key Point Recall (KPR)" to calculate the coverage between the generated and human-written context. Within this paper, semantic coverage is assessed alongside factual consistency as a dimension for evaluating generated text.
