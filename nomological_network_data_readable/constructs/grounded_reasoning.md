# Grounded reasoning
**Type:** Construct  
**Referenced in:** 4 papers  
**Also known as:** Context-grounded reasoning  

## State of the Field

Across the provided literature, grounded reasoning is a construct defined in several distinct ways, with the common element being that reasoning is based on provided evidence rather than abstract knowledge alone. One line of work frames it as "context-grounded reasoning," or the ability to generate answers consistent with a textual context like a document or dialogue history, and operationalizes it through tasks such as comprehension-based QA and dialogue reasoning. Another perspective, found in multiple papers, defines it as a capacity for reasoning based on "simulated, first-hand experience within an environment's dynamics," which enables a form of trial and error. A third definition, specific to vision-language models, describes it as the ability to base responses on "actual visual evidence" from an input image. This is explicitly contrasted with the behavior of "producing outputs containing inaccurate or fabricated details." The construct is noted as a target of evaluation in the surveyed work.
