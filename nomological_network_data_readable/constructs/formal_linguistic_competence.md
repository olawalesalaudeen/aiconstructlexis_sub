# Formal linguistic competence
**Type:** Construct  
**Referenced in:** 3 papers  

## State of the Field

In the provided literature, formal linguistic competence is defined as the ability to generate fluent and grammatically correct text based on statistical patterns learned from training data. This competence is explicitly framed as a surface-level capability reflecting 'data' memory, rather than deep semantic understanding. As one paper states, it "mainly comes from statistical learning during training on large text corpora" (EcoLoRA: Communication-Efficient Federated Fine-Tuning of Large Language Models). To operationalize and measure this construct, the surveyed work uses specific benchmarks, namely BLiMP and SyntaxGym. Additionally, formal linguistic competence is noted to correlate with brain alignment.
