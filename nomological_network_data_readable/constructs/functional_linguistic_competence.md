# Functional linguistic competence
**Type:** Construct  
**Referenced in:** 3 papers  

## State of the Field

Functional linguistic competence is described in the provided literature as an ability to understand deeper meanings in language that goes beyond parsing surface text. The single available definition characterizes it as involving "semantic integration and reasoning about roles and relationships" and reflecting a model's "knowledge' memory." This construct is operationalized and assessed using a range of benchmarks that target distinct reasoning types. Specifically, papers use ARC-Easy and ARC-Challenge to measure world knowledge, SocialIQA for social reasoning, PIQA for physical reasoning, and WinoGrande and HellaSwag for commonsense reasoning as indicators of this competence. Despite this operationalization, one source notes that the concept "is still unclear and under debate." The construct is also reported to correlate with Brain alignment.
