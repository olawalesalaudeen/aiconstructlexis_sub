# Complexity
**Type:** Construct  
**Referenced in:** 6 papers  

## State of the Field

Across the surveyed literature, Complexity is consistently defined as the level of "sophistication, depth, or intricacy" in a language model's generated response. It is commonly framed as one of several attributes for evaluating model outputs, appearing alongside helpfulness, correctness, coherence, and verbosity in reward frameworks. One paper specifies its use as a "rating dimension in preference annotation." To operationalize this construct, researchers use specific measurement instruments; the provided data shows that Complexity is measured by both the HelpSteer and HelpSteer2 datasets. The construct is also reported to correlate with Helpfulness.
