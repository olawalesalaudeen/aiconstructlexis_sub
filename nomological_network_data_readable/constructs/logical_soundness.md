# Logical soundness
**Type:** Construct  
**Referenced in:** 2 papers  
**Also known as:** Logical alignment, Logical plausibility  

## State of the Field

The degree to which intermediate sub-queries or reasoning steps are logically consistent with the prior reasoning context.
