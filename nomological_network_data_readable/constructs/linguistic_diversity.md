# Linguistic diversity
**Type:** Construct  
**Referenced in:** 4 papers  

## State of the Field

Based on the available data, linguistic diversity is defined as a model's capacity to generate or recognize multiple valid phrasings that convey the same semantic content. This concept is framed as a reflection of natural language variation, where a single idea can have multiple valid expressions. For example, one paper illustrates this with the phrases "There are 12 inches in 1 foot" and "There are 12 inches in each foot," which are used to highlight "nuanced token-level dependencies" that some methods overlook (SSA: Semantic Contamination ofLLM-Driven Fake News Detection). The concept is discussed in the context of Supervised Fine-Tuning (SFT), which is described as having an "inability to capture inter-token relationships and linguistic diversity." Consequently, preserving this "expression diversity" is presented as a goal for model training.
