# Multimodal perception
**Type:** Construct  
**Referenced in:** 8 papers  

## State of the Field

Across the surveyed literature, multimodal perception is consistently defined as a latent ability for models or agents to comprehend and interpret an environment by integrating information from multiple modalities, most commonly visual and textual inputs. Some work frames this as a foundational or "most basic ability" for large multimodal agents, enabling awareness of status, surroundings, and objects. The application of this construct varies, from providing a "detailed description of the agent status and environmental features" for embodied agents to enabling subject-driven image generation from interleaved images and text. A more specialized use case involves extracting information from charts by recognizing "visual elements, such as colors, structures, and spatial relationships." The field operationalizes and measures this construct through agent performance on a wide array of evaluation benchmarks. These measurement instruments include MME, MMMU, MMBench, SEED, GQA, ScienceQA, MathVista, AI2D, ChartQA, OCRBench, TextVQA, DocVQA, MMVP, and RealWorldQA.
