# Transition awareness
**Type:** Construct  
**Referenced in:** 2 papers  
**Also known as:** Transition naturalness, Transition-awareness  

## State of the Field

The latent ability to detect dialogue mode changes and adapt responses appropriately when conversations shift between task-oriented dialogue and chitchat.
