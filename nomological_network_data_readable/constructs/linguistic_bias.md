# Linguistic bias
**Type:** Construct  
**Referenced in:** 2 papers  
**Also known as:** Language-specific performance  

## State of the Field

The latent capability of a multilingual LLM to perform accurately in a particular language, reflecting differences in training data coverage and model fluency.

## Sources

**[Membership and Memorization inLLMKnowledge Distillation](https://aclanthology.org/2025.emnlp-main.1016.pdf) (as "Language-specific performance")**

**[Languages Still Left Behind: Toward a Better Multilingual Machine Translation Benchmark](https://aclanthology.org/2025.emnlp-main.1019.pdf)**
> The tendency for a model to exhibit differential performance on a task depending on the linguistic modality of the input, even when the content is similar.
