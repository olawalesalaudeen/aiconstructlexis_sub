# Low-rank structure
**Type:** Construct  
**Referenced in:** 2 papers  

## State of the Field

The property of certain weight matrices (particularly delta weights) where most of the information is concentrated in a small number of singular vectors, enabling efficient approximation via low-rank decomposition.
