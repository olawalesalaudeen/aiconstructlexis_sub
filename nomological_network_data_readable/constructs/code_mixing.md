# Code-mixing
**Type:** Construct  
**Referenced in:** 2 papers  
**Also known as:** Code-mixing capability, Code-mixing comprehension  

## State of the Field

The latent ability to understand and process multilingual text in which languages are switched or mixed within a single utterance or conversation.
