# Token diversity
**Type:** Construct  
**Referenced in:** 3 papers  
**Also known as:** Token Diversity  

## State of the Field

Across the provided literature, token diversity is consistently defined as the degree of variation in the tokens a model produces, with an emphasis on avoiding repetitive outputs. The concept is framed as the extent to which generated text uses a "varied token distribution" ("A Closer Look at Machine Unlearning for Large Language Models") and reflects the "richness of the output distribution" ("Masked Diffusion Models are Secretly Time-Agnostic Masked Models and Exploit Inaccurate Categorical Sampling"). This construct is operationalized and measured using specific metrics, with one paper explicitly stating that "Token Entropy (TE) measures the diversity of tokens in the model’s output" ("A Closer Look at Machine Unlearning for Large Language Models"). Token diversity is positioned as a component of generation quality, with one study noting that a decrease in this attribute can make evaluations based solely on generative perplexity "somewhat unfair" ("Masked Diffusion Models are Secretly Time-Agnostic Masked Models and Exploit Inaccurate Categorical Sampling").
