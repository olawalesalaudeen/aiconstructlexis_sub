# Parametric memory
**Type:** Construct  
**Referenced in:** 4 papers  

## State of the Field

Across the surveyed literature, parametric memory is consistently defined as the knowledge stored implicitly within a language model's learned parameters, as opposed to knowledge accessed from an external source. This internal knowledge is described as being formed during pre-training and containing "commonsense and factual knowledge" as well as "rules" ("Adaptive Chameleon  or Stubborn Sloth...", "Grokking of Implicit Reasoning..."). Some work frames this as the "LLM’s internal belief" ("Adaptive Chameleon  or Stubborn Sloth..."). It is what models rely on when operating "without retrieval" ("Making Retrieval-Augmented Language Models Robust to Irrelevant Context"). However, multiple sources also highlight its limitations. For instance, parametric memory is noted to be potentially "inaccurate or become outdated" due to its static nature or misinformation in the training data ("Adaptive Chameleon  or Stubborn Sloth..."). Furthermore, some research suggests that large language models can be "deficient in implicit reasoning with their parametric memory" ("Grokking of Implicit Reasoning...").
