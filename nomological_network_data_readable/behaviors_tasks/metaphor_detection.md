# Metaphor detection
**Type:** Behavior  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, metaphor detection is defined as the observable task of identifying whether a sentence contains a metaphor. This behavior is operationalized as a prediction task where a model assigns a corresponding label to a given sentence. The available sources situate this task within the broader study of figurative language understanding, where it is treated as a form of figurative speech. It is frequently studied alongside other complex linguistic phenomena, such as sarcasm and hyperbole. One paper also groups the detection of metaphors with the detection of harmful language patterns like sexism.
