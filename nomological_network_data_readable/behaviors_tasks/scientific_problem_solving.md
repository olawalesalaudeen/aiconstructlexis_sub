# Scientific problem solving
**Type:** Behavior  
**Referenced in:** 5 papers  

## State of the Field

Within the provided literature, scientific problem solving is defined as the end-to-end behavior of receiving a complex problem from a natural science domain and producing a complete, functional coded solution. This framing focuses on the capability of models "to generate code for solving real scientific research problems" (SciCode: A Research Coding Benchmark Curated by Scientists). The field operationalizes and measures this behavior using several named instruments. Papers report using SciBench and OlympiadBench to assess scientific problem solving. The paper that introduces the concept's definition also developed a benchmark for its evaluation, SciCode, which is described as containing problems from 16 diverse domains. The behavior is also studied in relation to domain-specific knowledge.
