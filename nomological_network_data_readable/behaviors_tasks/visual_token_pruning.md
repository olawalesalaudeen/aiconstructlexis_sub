# Visual token pruning
**Type:** Behavior  
**Referenced in:** 2 papers  
**Also known as:** Visual token compression, Token pruning  

## State of the Field

The process of removing or compressing visual tokens from an image representation in a vision-language model to reduce computational load while preserving task-relevant information.
