# Tabular classification
**Type:** Behavior  
**Referenced in:** 4 papers  

## State of the Field

Tabular classification is defined in the provided literature as the task of predicting a categorical label for a data point from a set of structured, tabular features. The task is operationalized and measured using statistical methods, with one paper employing the Wilcoxon signed-rank test for evaluation. It is studied in relation to in-context and few-shot learning. Research on this task also investigates methods for improving sample efficiency, with one study reporting that test-time training "significantly reduces the required sample size for tabular classification" ("Test-Time Training Provably Improves Transformers as In-context Learners").
