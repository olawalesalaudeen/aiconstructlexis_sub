**Type:** Behavior

## State of the Field

Based on the provided data, misinformation generation is defined as the creation of convincing but deceptive content. The available sources frame this behavior in the context of its use for social media campaigns or to obscure authentic discourse, with one paper noting that "synthetic text can fuel misinformation." This framing is supported by experimental findings mentioned in one source, which demonstrate that "AI-generated content can create convincing but deceptive social media campaigns."
