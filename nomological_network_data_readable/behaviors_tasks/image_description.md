# Image description
**Type:** Behavior  
**Referenced in:** 9 papers  

## State of the Field

Across the surveyed literature, image description is consistently defined as the behavior of generating a textual account of an image's visual content. The prevailing usage frames this as describing entities, attributes, and their relationships, with one paper specifying that it involves "steps derived solely from visual information" ("Correcting Negative Bias in Large Language Models through Negative Attention Score Alignment"). A more specific framing distinguishes this from OCR, which is reserved for text-rich parts of an image. Some work extends the definition to encompass higher-level cognition, where models are expected to "understand and describe the story in the image through high-level cognitive reasoning" ("CharacterBox: Evaluating the Role-Playing Capabilities ofLLMs in Text-Based Virtual Worlds"). To operationalize this behavior, researchers use measurement instruments such as Q-Bench. The task is studied in relation to visual understanding, and its failure modes are frequently associated with hallucination and object hallucination.
