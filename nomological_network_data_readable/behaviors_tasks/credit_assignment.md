# Credit assignment
**Type:** Behavior  
**Referenced in:** 6 papers  

## State of the Field

Across the surveyed literature, credit assignment is consistently described as the ability to identify which specific prior actions in a multi-step sequence are responsible for a subsequent success, failure, or reward. This construct is discussed in the context of language agents, where it is framed as understanding "where the agent made mistakes," and in the context of reward models, where it involves pinpointing erroneous steps in a reasoning chain. The challenge is noted as being particularly difficult when rewards are delayed or when supervision is only provided for the final outcome. To operationalize and test this ability, researchers use tasks such as Chess, Endgames, Maze, and Text-Nav, which require an algorithm to "assign credit to good actions rather than a lucky starting position." Some work also connects this construct to "root cause analysis abilities." Finally, credit assignment is reported to be a cause of learning efficiency.
