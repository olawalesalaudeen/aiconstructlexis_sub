# Graph classification
**Type:** Behavior  
**Referenced in:** 4 papers  

## State of the Field

Graph classification is consistently described as the task of assigning a label to an entire graph. The definitions provided are largely in agreement, with one framing it generally as assigning "a categorical label to an entire graph" (PLLuM-Align:Polish Preference Dataset for Large Language Model Alignment) and another offering a more specific operationalization of assigning "class labels to graph inputs represented as projected embeddings" (Vector-ICL: In-context Learning with Continuous Vector Representations). This behavior is commonly studied in conjunction with other graph-related tasks, namely node classification and link prediction. The provided materials describe different approaches to this task; one paper mentions training "non-linear projectors on the graph classification dataset" (Vector-ICL: In-context Learning with Continuous Vector Representations), while another discusses reformulating the task as a "textual reasoning problem" (PLLuM-Align:Polish Preference Dataset for Large Language Model Alignment).
