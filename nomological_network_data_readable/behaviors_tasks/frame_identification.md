# Frame identification
**Type:** Behavior  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, frame identification is presented as a behavior central to frame semantic parsing. The single source paper consistently defines it as the task of selecting the correct semantic frame for a target word within a specific context. This process is described as determining the semantic frame that is "evoked by a target word in context" and is framed as a "core challenge" in its domain. To operationalize and measure this behavior, the literature uses the FrameNet 1.7 dataset. Performance on the task is evaluated using accuracy, as indicated by evidence comparing model performance on both the FN 1.5 and FN 1.7 datasets.
