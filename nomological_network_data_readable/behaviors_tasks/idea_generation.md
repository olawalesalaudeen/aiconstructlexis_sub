# Idea generation
**Type:** Behavior  
**Referenced in:** 12 papers  
**Also known as:** Brainstorming, Topic brainstorming, Feature proposal, Alternative uses generation, Research proposal generation  

## State of the Field

Across the provided literature, idea generation is broadly defined as the observable process of producing novel concepts or solutions. A prevalent framing is "brainstorming," which involves generating a list of ideas or suggestions in response to a prompt, with some work specifying this as generating diverse topics for argumentative texts. This behavior is sometimes evaluated for "fluency and relevance." Another line of work connects idea generation to divergent thinking, operationalizing it through tasks like "alternative uses generation," where a model lists novel uses for an everyday object, a task associated with the Alternate Uses Test (AUT). More structured applications also appear, such as "feature proposal," where an LLM agent acts as an expert to generate conceptual ideas, and "research proposal generation," which involves producing detailed documents with specific sections like a problem statement and experiment plan. The provided data indicates that the behavior of idea generation is measured using LLM-as-a-judge.
