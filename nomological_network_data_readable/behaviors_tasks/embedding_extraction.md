# Embedding extraction
**Type:** Behavior  
**Referenced in:** 2 papers  
**Also known as:** Region embedding extraction  

## State of the Field

The process of obtaining vector representations of input samples from a specific layer of a neural network, used to analyze semantic relationships in the model's representation space.
