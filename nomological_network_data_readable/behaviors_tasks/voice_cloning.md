# Voice cloning
**Type:** Behavior  
**Referenced in:** 3 papers  
**Also known as:** Zero-shot text-to-speech, Speaker-specific adaptation  

## State of the Field

Generating speech in a new speaker's voice using only a short audio prompt from that speaker, without any speaker-specific training.

## Sources

**[SurveyGen: Quality-Aware Scientific Survey Generation with Large Language Models](https://aclanthology.org/2025.emnlp-main.137.pdf) (as "Zero-shot text-to-speech")**
> Generating speech in a new speaker's voice using only a short audio prompt from that speaker, without any speaker-specific training.

**[SurveyGen: Quality-Aware Scientific Survey Generation with Large Language Models](https://aclanthology.org/2025.emnlp-main.137.pdf)**
> The specific task of synthesizing speech that mimics the voice of a particular person, often in a zero-shot setting from a brief audio sample.

**[SolEval: Benchmarking Large Language Models for Repository-level Solidity Smart Contract Generation](https://aclanthology.org/2025.emnlp-main.219.pdf) (as "Speaker-specific adaptation")**
> Reducing word error rates more effectively when context examples come from the same speaker as the target audio, especially at intermediate shot counts.
