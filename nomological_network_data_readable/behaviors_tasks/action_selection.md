# Action selection
**Type:** Behavior  
**Referenced in:** 4 papers  

## State of the Field

Based on the provided literature, action selection is defined as the process of choosing an optimal action from a set of candidates. This process is described as being guided by a look-ahead or evaluation mechanism, with one paper specifying that it "enables look-ahead planning to select optimal actions during inference" ("IIET: Efficient Numerical Transformer via Implicit Iterative Euler Method"). The concept appears in the context of agent LLMs, where look-ahead simulation is used to guide the agent's choices. In addition to these mechanisms, the behavior is also reported to be influenced by the construct of commonsense understanding.
