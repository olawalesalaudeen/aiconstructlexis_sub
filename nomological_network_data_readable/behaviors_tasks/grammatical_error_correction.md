# Grammatical error correction
**Type:** Behavior  
**Referenced in:** 7 papers  

## State of the Field

Across the provided literature, Grammatical error correction (GEC) is predominantly characterized as an observable task or process of automatically detecting and correcting errors in a text to produce a more accurate version. This is frequently operationalized as a form of monolingual machine translation, where, as one paper states, GEC is treated as a task "where erroneous text is rewritten into corrected text." The specific corrections are described as involving "insertion, deletion, re-ordering, and substitution operations." While the name emphasizes grammar, some sources expand the scope to include correcting "spelling, and semantic errors" as well as "awkward phrases." A less common framing, found in one paper, defines GEC as a "latent ability," though the prevailing usage across the data centers on the observable task. The provided materials also indicate that GEC is studied in relation to `Evaluation` and `Multilingual ability`.
