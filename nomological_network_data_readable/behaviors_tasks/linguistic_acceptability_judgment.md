# Linguistic acceptability judgment
**Type:** Behavior  
**Referenced in:** 6 papers  
**Also known as:** Linguistic acceptability  

## State of the Field

Across the provided literature, linguistic acceptability judgment is consistently described as the task of determining if a given sentence is grammatically correct or acceptable. The definitions are largely in agreement, though one specifies this judgment is relative to a "native speaker," while another frames it as an "observable task of classifying a sentence as grammatically acceptable or not." This behavior is operationalized and measured using specific benchmarks. The most frequently mentioned instrument is CoLA, which multiple papers situate within the GLUE benchmark. The GLUE-X benchmark is also cited as a tool used to measure this behavior.
