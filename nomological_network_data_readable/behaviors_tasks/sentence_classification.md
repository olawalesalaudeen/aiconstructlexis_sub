# Sentence classification
**Type:** Behavior  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, sentence classification is defined as the task of categorizing sentences based on their syntactic structure. The specific categories mentioned are simple, complex, compound, or compound-complex. This behavior is operationalized through a supervised learning approach where models are fine-tuned and evaluated. For instance, one paper describes an approach where "six transformer-based models and two smaller LLMs" were fine-tuned on a training set and then assessed on a test set (Complex Numerical Reasoning with Numerical Semantic Pre-training Framework). The output of this process is described as "a list of correctly classified sentences with labels."
