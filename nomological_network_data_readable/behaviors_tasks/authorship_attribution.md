## State of the Field

The prevailing usage of authorship attribution in the provided literature is as a behavior involving the classification of a text's origin, most commonly as a binary distinction between human-written and machine-generated content. This task appears under several names, including 'LLM text detection', 'AI-generated text detection', and 'Machine-generated text detection', with one paper noting that methods have "traditionally formulated the task as a binary classification problem" (CAT: Causal Attention Tuning For Injecting Fine-grained Causal Knowledge into Large Language Models). A smaller set of papers defines more specific tasks, such as determining if two texts share the same author, verifying if a text was written by a specific known author ('Authorship verification'), or identifying which particular LLM generated a text ('Text origin detection'). This behavior is operationalized and measured using benchmarks like the MAGE dataset, on which models are trained and evaluated. The technical approaches mentioned for this classification include leveraging features from logit distributions or from "entity graphs and contextual representations" (T2: An Adaptive Test-Time Scaling Strategy for Contextual Question Answering). The literature also discusses related but distinct behaviors, such as 'Authorship mimicking' (replicating a style) and 'Authorship obfuscation' (concealing a style). The ambiguity this task seeks to resolve is linked to the concept of 'Accountability', which is described as being diffused when authorship is unclear, a phenomenon quantified by one paper as 'Authorship entropy'.
