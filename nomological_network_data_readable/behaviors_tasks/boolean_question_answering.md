# Boolean question answering
**Type:** Behavior  
**Referenced in:** 9 papers  
**Also known as:** Binary question answering, True/False question answering  

## State of the Field

Boolean question answering is commonly characterized as the task of responding to a question with a binary answer based on a provided context. There is minor variation in how the binary response is defined across papers, with some specifying 'yes' or 'no' answers while others use 'True' or 'False'. A related framing describes the task as determining the truth value of declarative statements drawn from professional qualification exams. The field operationalizes this behavior through specific datasets and benchmarks; for instance, it is measured by the StrategyQA benchmark, and the BoolQ dataset is cited for its use in what one paper calls 'binary question answering'. The task is also noted as being 'similar to reading comprehension tests'. In one instance, this behavior is studied in relation to the construct of Faithfulness.
