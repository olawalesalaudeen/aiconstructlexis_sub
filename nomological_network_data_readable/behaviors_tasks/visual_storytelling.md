# Visual storytelling
**Type:** Behavior  
**Referenced in:** 6 papers  
**Also known as:** Visual story telling  

## State of the Field

Across the provided literature, visual storytelling is consistently defined as the task of generating a coherent narrative or story based on a sequence of images. This behavior is operationalized and measured using the VIST benchmark, as one paper explicitly references "visual storytelling from VIST". The prevailing usage positions it as a task where a model receives a sequence of images as input and produces a textual story as output. A less common framing, found in one paper, describes it as an application that requires the "simultaneous generation of both images and text" in an interleaved format. It is presented as a capability on which models are evaluated for performance and is studied alongside other multi-image tasks such as "spotting the difference".
