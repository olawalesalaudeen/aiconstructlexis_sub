# Example generation
**Type:** Behavior  
**Referenced in:** 4 papers  

## State of the Field

Within the provided literature, example generation is characterized as the behavior of producing synthetic sentences to illustrate a specific word sense from a dictionary definition. This task is operationalized by using a large language model as a generator to produce multiple "distinct example sentences that demonstrate the target sense," as detailed in one study ("Context Reasoner..."). The stated goal is for LLMs to "generate diverse example sentences for dictionary senses." The study of this behavior is associated with the phenomenon of hallucination. Specifically, one paper reports that "gross hallucinations" are found when prompting a model to generate new examples, indicating this is a co-occurring issue observed during the task.
