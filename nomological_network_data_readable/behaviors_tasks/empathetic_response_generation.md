# Empathetic response generation
**Type:** Behavior  
**Referenced in:** 3 papers  

## State of the Field

Within the provided literature, empathetic response generation is defined as the behavior of producing supportive and emotionally appropriate replies in dialogue. This task is framed as requiring an understanding of both the speaker's emotional state and its causal basis. One documented approach aims to generate these responses by "effectively integrating contextual and causal information from the dialogue history" (Diagram-Driven Course Questions Generation). The performance of this behavior is operationalized and measured using the RECCON dataset. Additionally, language proficiency is presented as a factor that causes empathetic response generation.
