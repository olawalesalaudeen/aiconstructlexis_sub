# Resource collection
**Type:** Behavior  
**Referenced in:** 3 papers  
**Also known as:** Resource harvesting  

## State of the Field

Resource collection is a behavior conceptualized in at least two distinct ways across the provided sources. One framing defines it as a physical task where an agent gathers materials from an environment, such as "mining or gathering materials from terrain blocks to obtain items such as wood, stone, coal, iron, diamond" ("Mars: Situated Inductive Reasoning in an Open-World Environment"). This operationalization focuses on the direct interaction with the environment, where an agent uses a tool to "mine items and obtain resources." A different framing, using the term "resource harvesting," treats the behavior as a strategic decision in a social context. In this view, it is "the observable action of an agent deciding on and extracting a specific quantity of a shared resource from the common pool" ("Cooperate or Collapse: Emergence of Sustainable Cooperation in a Society of LLM Agents"). This is operationalized within the GOVSIM simulation, where a "society of AI agents must collectively balance exploiting a common resource with sustaining it for future use." Thus, the concept is used to describe both low-level environmental interactions and higher-level strategic actions in multi-agent resource management scenarios.
