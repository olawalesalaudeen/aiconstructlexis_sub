**Type:** Behavior  
**Referenced in:** 4 papers  

## State of the Field

In the surveyed literature, 'Audio question answering' is characterized as the behavior of generating answers to natural language questions about audio content. The single available definition specifies that this task is based on both the audio input and accompanying text. The primary way this behavior is operationalized and measured in the provided sources is through the ClothoAQA dataset. One paper describes this dataset as comprising audio samples paired with "crowdsourced questions and corresponding answers" ("CalligraphicOCRforChinese Calligraphy Recognition"). Performance on the task is assessed using accuracy, with one study reporting specific accuracy scores for a model.
