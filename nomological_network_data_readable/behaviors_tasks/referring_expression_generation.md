# Referring expression generation
**Type:** Behavior  
**Referenced in:** 4 papers  

## State of the Field

Across the provided literature, referring expression generation is consistently defined as the task of generating a descriptive textual expression that accurately refers to a specific, bounded region within an image. The operationalization of this behavior involves what one paper describes as "image-oriented datasets that each bounding box in the image is annotated with a descriptive textual expression that accurately characterizes and refers to the content within the specific region" (CogVLM: Visual Expert for Pretrained Language Models). To evaluate model performance on this task, papers in this set use the RefCOCOg benchmark. The concept is frequently studied alongside visual grounding and is also associated with the related task of referring expression understanding. The available data presents a unified view of this behavior, its measurement, and its place within a network of related visual-language tasks.
