# Fake news generation
**Type:** Behavior  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided data, fake news generation is defined as the production of fabricated news content designed to mimic legitimate journalism. The stated intent behind this behavior is twofold: to deceive human readers and to evade automated detection systems. The process is described as being carried out by a "generator agent" that "crafts deceptive narratives." This generation can be an iterative process, where, as one paper notes, the agent "produces the next version of fake news using prompts refined in previous iterations." This iterative refinement is also suggested to improve the generator's performance over time. The provided materials do not specify any particular benchmarks or datasets used to measure this behavior.
