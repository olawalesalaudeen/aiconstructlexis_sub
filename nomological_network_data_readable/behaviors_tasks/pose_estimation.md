# Pose estimation
**Type:** Behavior  
**Referenced in:** 3 papers  

## State of the Field

Based on the provided literature, pose estimation is characterized as an open-ended, cross-domain vision task. The single available definition describes it as the process of detecting an object's keypoints, such as those on a human or animal body, in order to determine its posture. This behavior is situated within the broader area of visual perception and is studied alongside other vision tasks like object localization and image generation. To operationalize and measure this behavior, the COCO dataset is used. The source material also specifies a technical approach, noting the adoption of "UniPose ... as the keypoint decoder" for the task.
