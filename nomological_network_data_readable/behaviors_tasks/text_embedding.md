# Text embedding
**Type:** Behavior  
**Referenced in:** 6 papers  

## State of the Field

Across the provided literature, text embedding is consistently defined as the process of mapping or producing dense vector representations from text inputs. The purpose of these vectors is described as capturing the "semantic meaning of text" for use in downstream methods, with one definition specifically highlighting similarity-based applications. One paper notes that these vector representations "serve as the foundation for many natural language processing (NLP) tasks." Large language models are explicitly mentioned as being explored as "powerful text embedders." A specific operational detail mentioned in one source is that models can "take text inputs and output 3072-dimensional vectors." To evaluate this behavior, the provided papers use the MTEB benchmark.
