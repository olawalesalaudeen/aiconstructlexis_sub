# Interactive segmentation
**Type:** Behavior  
**Referenced in:** 3 papers  

## State of the Field

Based on the available data, interactive segmentation is defined as a task involving the segmentation of objects in an image through the active incorporation of user inputs. The operationalization of this behavior is central to its definition, with user inputs specified as points, boxes, or scribbles that guide the segmentation process. One source describes this as a method that "separates objects by actively integrating user inputs" ("Interfacing Foundation Models' Embeddings"). The task is also situated within the context of model training, where it is noted that training for interactive segmentation can occur alongside other tasks, such as image-text retrieval, without mutual interference.
