# Chart summarization
**Type:** Behavior  
**Referenced in:** 4 papers  

## State of the Field

Across the provided literature, "Chart summarization" is consistently defined as the behavior of generating a textual summary or analysis from a chart's content, as opposed to answering a single factual question. The definitions describe the output as either a general "textual summary or analysis" ("ChartMoE...") or more specifically as a "concise, open-ended textual summary that describes the key trends, insights, or information" ("LegalSearchLM..."). The open-ended nature of the task is a recurring theme, with one paper characterizing it as a type of question "lacking a single definitive answer" ("LegalSearchLM..."). One study situates chart summarization as a component of the broader category of "Chart Reasoning" and notes that code can enable "in-depth summarization" ("ChartMoE..."). The provided data does not specify the benchmarks or evaluation procedures used to operationalize or measure this behavior.
