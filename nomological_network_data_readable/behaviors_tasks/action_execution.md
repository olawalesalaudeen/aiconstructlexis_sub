# Action execution
**Type:** Behavior  
**Referenced in:** 4 papers  

## State of the Field

Based on the provided literature, action execution is defined as an agent's observable performance of a specific action from its available action space within an environment. This behavior is described as involving an interaction module that executes actions and records the processes, as detailed in "ADAM: An Embodied Causal Agent in Open-World Environments". To operationalize and assess this behavior, the field employs measurement instruments like T-Eval. The study of action execution is also situated alongside other agentic capabilities, as it is shown to be related to both planning and commonsense knowledge. Thus, the concept is framed as the practical, observable component of an agent's behavior, which is evaluated through specific tools and considered in conjunction with related cognitive functions.
